Memory Stick:

AIBO's use a number of different files to store information for Actions, Motion, LED's, etc...   This section provides technical information for parsing such files.
  • Object Database Files (*.oda) - Archive files of motion/led/sound elements on AIBO memory stick.
  • Middleware Catalog File (*.mwc) - Index linking motion/led/sound elements for all skits.
  • Action Config Files (action.cfg, action.ext, action.ams) - Provides meaningful skit names in RCode & YART.

Skitter:
  • Action Files (*.act) - Associates motion, led & wave files together for single skit.
  • Motion Files (*.mtn) - AIBO Motion files.
  • LED Files (*.led) - AIBO LED files.
  • MIDI Sound Files (*.mid) - AIBO compatible MIDI sound files.
  • Wave Sound Files (*.wav) - AIBO compatible Wave sound files.
  • Workbook Files (*.aib) - All-in-one file containing motion, led & wave elements for single skit.
  • User Action Files (*.uac) - Text based file for associating Motion, LED & Wave files.

Master Studio:
  • Master Studio Album Files (*.alb) - Contains list of actions/skits.
  • Master Studio Action Files (*.act) - Associates Motion, LED & sound files together for single skit.
  • Master Studio Behavior Files (*.be) - Behavior flow-chart files.
