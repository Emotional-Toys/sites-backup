[AiboHack] Terminator AIBO

Terminator AIBO

Terminator is based on the movie Terminator 2. He is basically a good guy.


(artwork and usage instructions also by teamdax)

Supported AIBOs

  • - ERS-210 is supported.
  • - ERS-220 is supported.
  • - ERS-311/312 is supported.

Installation

Required Hardware:

  • An ERS-210 model AIBO OR
  • An ERS-220 model AIBO OR
  • An ERS-31x model AIBO (311 Latte, 312 Macaron or 31L Pug)

  • A Pink Programmable Memory Stick (AKA "PMS" or RCode stick) - ERA-MS008 or ERA-MS016, $34->$39
  • A memory stick reader to install the software (Use only Sony brand stick readers)

Installation:

  • Put an available PMS stick into your computer's stick reader
  • Select any folders you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. Do not format the stick.
  • Download the proper RCodePlus runtime for the model of your AIBO.
    • For the ERS-210: RCode Plus 2.52 'A' (210)
    • For the ERS-220: RCode Plus 2.52 'A' (220)
    • For the ERS-31x: RCode Plus 2.52 'A' (31x)
    • (Save this .ZIP file on your computer. You may want to use it again later.)
  • Unzip this RCodePlus file to the root of your PMS memory stick.
  • Download the Terminator Personality: terminator.zip (1/2 MB)
  • Unzip that file on top of your RCODE/PMS stick (replacing any files it asks about).

  • Put the memory stick inside your AIBO and start it up.
  • Play with your AIBO. Follow the instructions (see below).

Usage


  • New feature: he can walk and talk at the same time. You can also pause him by saying "STOP" (more details below).

  • Taking a photo: this is a two-stage process. First press the head button. When you release the head button he will make a suitable sound, and quickly take a photo. You can also use the voice command "Take a picture"
  • Singing: –Terminator will sing the theme song at boot up. Tell him to "Sing a song" to get him to repeat it at any time
  • Interaction/ Autonomy: Terminator is very active, walking and talking.
  • Voice commands: He should also respond to most of the standard voice commands with customs sounds and actions. If he doesn't want to listen, wait till he stops walking.
    Try all possible voice commands for some amusing sayings.

  • Stop: If Terminator gets into trouble, or if you just want him to sit perfectly still for a while, say "STOP". He will sit down, and remain still (and quiet) until you press his head button.

  • If he seems to freeze, try giving him a verbal command. If that doesn't work, press his head button. Finally if that doesn't work tip him on his side.

Customization

You can customize the personality using YART. To run YART on the personality, just open the "\CUSTOM" folder on the memory stick and run the YART.EXE program in there.

Please refer to the general YART tutorial if you have never used YART before.
