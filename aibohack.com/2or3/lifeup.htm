AiboLife Upgrades

Checking sub-version numbers

ABrowser23 (version 3.2 or later) will show the revision number for some versions of AiboWare. It will recommend you upgrade if it detects an old version.
Original versions are usually numbered "1.0". Fixed versions are usually numbered "1.1". This can get confusing when talking about AiboLife 1 and 2, and then include version numbers on top of that.

Aibo Life 1 (ERF-210AW01) - for the ERS-210

For the old AiboLife (1) for the ERS-210. The last release is revision 1.1 AiboLife 1 Plus includes the latest version of AiboLife 1 (release 1.1).
This is not a critical upgrade (unless you want the features of AiboLife 1 Plus)

Motion Training Bug

The original AiboLife 2 and AiboLife 310 (release 1.0) has a bug with motion recording. Digression: In Japan, Sony recalled the memory sticks and offered free fixes. In the US, they documented the behavior of the bug with no fix, so you now know how to live with it.
Considering US owners pay approximately $90 for AiboLife2 and Japanese owners pay under $70 for the same software - one might expect US owners to get better support ! End of digression

Aibo Life 2 (ERF-220AW01) - for the ERS-210 or ERS-220

The original AiboLife 2 (release 1.0) has a bug with motion recording. To upgrade to the fixed release (1.1), download one of the following files and unzip it to the root of your memory stick. Your unzip program will ask you to replace two files, answer yes to both.
  • For English AiboLife 2: lifeup_ERF220AW01E.zip
  • For Japanese AiboLife 2: lifeup_ERF220AW01J.zip
This is not a critical upgrade, but recommended if you use the motion recording feature ("Training Mode")

Aibo Life 310 (ERF-310AW01) - for the ERS-311, ERS-312, ERS-31L

The same motion recording bug exists in the version of AiboLife for the 31x. To upgrade to the fixed release (1.1), download one of the following files and unzip it to the root of your memory stick. Your unzip program will ask you to replace two files, answer yes to both.
  • For English AiboLife 310: lifeup_ERF310AW01E.zip
  • For Japanese AiboLife 310: lifeup_ERF310AW01J.zip
This is not a critical upgrade, but recommended if you use the motion recording feature ("Training Mode")
