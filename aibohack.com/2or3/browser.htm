ABrowser23 - Main Features

ABrowser23 for the ERS-2x0 and ERS-31x
You must use the ERS-11X browser for the original model AIBOs. ABrowser23 works for the newer model AIBOs (ERS-210 and later).

The browser has many advanced features, but page will deal with the common browsing features.

Installation

The AIBO Browser is installed on your PC. It can be used to browse an actual memory stick, or "memory stick images" which are stored on your PC.
To install:
  • Download the latest Browser: browser23.zip
  • Unzip it to a folder on your PC (say C:\AIBO\TOOLS)
  • This will give you the Browser (ABrowser23.EXE) and a required DLL (virtaibo2.dll) which must be in the same directory.

Startup

When you run the Browser, it will search for the first drive with a memory stick image on it. You can abort this process and manually pick the drive you want to examine. You must open a "MEMSTICK.IND" file at the root of the proper memory stick (or memory stick image). If you do not see a "MEMSTICK.IND" file, you may need to make hidden files visible in Windows (see "Tools", "Folder Options" and the "View" tab).

Main Interface



The main interface has 3 lists. The first list is the "File" to browse (note: not all of these are actual files). Pick which "File" to browse, and the second and third list will fill in with the appropriate data.
Click around on anything of interest. Sometimes double-clicking on an item will cause it to be played back. Sometimes you need to press the "Play" button on the bottom.

Summary Page

This gives a summary of the current status of your AIBO.
Most of the information given in the summary page is also provided by the main "Examine" feature of AiboTool.

Photos (ERS-210 and AiboLife 1 only)



If you have AiboLife (W01) running on the ERS-210, you can take pictures by saying "Take a picture". The picture is stored on the memory stick in a non-standard format. You can use the APH2BMP utility to convert it to a .BMP, or use the Browser to display the image on your screen. The memory stick will hold at most 7 photographs. Select the "on memory stick" item in the browser to see the photos on the memory stick. Click on one of the Photos to see the photo and other information about your AIBO at the time the photo was taken.

"on PC"

The memory stick can only hold 7 photographs. When the 8th is taken, the oldest .APH file will be deleted. To save photographs, you will need to copy them to your PC. Create a "PHOTOS" folder on your PC (the "PHOTOS" folder must be in the same folder as ABrowser23.exe). Copy any APH files you like to the PHOTOS directory. Rename the files as necessary, but keep the .APH file extension.

Warning for Fun Pack owners

If you have the Sony Fun Pack, and you wish to use the Browser to view your photographs -- be sure to view and copy the .APH files before running the Fun Pack. When the Fun Pack starts, it will take the .APH files off the memory stick, save them away in the Fun Pack database, and then delete the .APH files. Once inside the Fun Pack, the photos can not be restored to .APH format.

"rhapsody" File



The "rhapsody" file gives detailed information about quality time as well as counts of key events in your AIBO's life. These counts get reset to zero when AIBO evolves to a new stage. Non-zero counts are listed for things that your AIBO has already done. Zero counts indicate things your AIBO has never done (at least not at this stage).

The general rule for raising AIBO is to keep counts of good events higher than counts of bad events. Examples of good events include: seeing the BALL or FAVorite color, being praised, being greeted ("Hello AIBO"). Examples of bad events include: seeing the UNFAV (hated) color, being hit or being bossed around ("Sit down").

Brain Surgery



Brain Surgery lets you change the stage and personality of your AIBO.

User ODA Files



You can browse the "user" ODA files directly from the memory stick. These files will not exist on the AiboLife stick (unless you have the Fun Pack) or the Hello AIBO stick.
Select a Motion, Sound or LED performance and press "Play" to see/hear the performance. For LED performances, they are played as musical tunes.

Examine Color Preferences

From the "View" menu, you can view the current color preferences of your AIBO. You can use AiboTool to change the color preferences.
