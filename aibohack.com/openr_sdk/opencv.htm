OpenCV Library - Test1

Port of Intel's OpenCV library

Source for library and simple test program: opencv001.zip.
The "readme.txt" file describes the new directory structure It can capture, save as bitmap (color or B&W), do the "canny" edge detection, do a laplacian ...
"make" at the root, and then copy MS contents to root of an SDK prepared memory stick.
Run it and it will save a simple drawing (rectangle, circles etc).

Press your AIBOs head hard and he will capture a new image, and save 4 .BMPs (color, B&W, canny, laplacian)

-- -- --

You can press his head many times to try different pictures. When done, take out the stick (the pause button doesn't work for me - not necessary), and view the .BMP files at the root of the stick
