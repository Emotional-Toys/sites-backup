OPEN-R SDK getting started advice

Overview

The official Sony Open-R SDK website has been discontinued. See this page for where to get some of the parts.

Requirements

Hardware Requirements:

  • An ERS-210 or ERS-220 ("A" models are optional)
  • NOTE: if you have an old ERS-210, you may need to upgrade the ROM. See the installation instructions.
  • One (or more) pink PMS/RCODE stick. Either in 8MB ($34) or 16MB (4000 Yen) sizes. Will not work with other sticks (eg: AiboLife or Hello AIBO).
  • A PC running Windows 2K or Windows XP (note: Linux/Unix machines are supported too, but this page focuses on the Windows installation)
  • A LAN setup for debugging (AIBO LAN card in your AIBO, WiFi LAN card in your Windows 2K/XP PC)

Skill Requirements:

  • You should be familiar with programming in C++. This is not the place to learn C++ or programming in general.
  • You should be generally familiar with Unix style build tools (On the PC, the SDK runs using Cygwin which provides a "Unix-like" command shell)

Skill Recommendations:

  • Familiarity with object-oriented programming helps.
  • Familiarity with a message passing communication system helps.

Getting in over your head

This is a very complicated and powerful SDK. It may not be your cup of tea. I recommend my 10 (or 11) steps below for getting into the SDK. This starts with the easy parts.

If you decide the OPEN-R SDK is not the right fit for you, but you still want to program your AIBO, you should look into writing RCODE programs. There are a variety of RCODE tools available (Master Studio, AbNet, YART or any text editor).
