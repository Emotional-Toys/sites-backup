Speed Racer (ERS-2x0 or ERS-7)

WARNING!!

Aibo Speed Racer uses the CMU "knee-walk". Using the head and body 'kicks' can be quite aggressive. The RoboCup dogs wear out their knees and necks faster than normal AIBOs because of these strange walks. Your AIBO may fall down or servo jam.
Use at your own risk! SpeedRacer will to turn your AIBO into a remote control toy. You can see what AIBO is seeing using the full resolution of the camera.


ERS-7 version with 416x320 video

Supported AIBOs

  • - ERS-210 is supported with a WiFi card.
  • - ERS-220 is supported with a WiFi card.
  • - ERS-7 is supported

Installation

Required Hardware:

  • An ERS-210[A] or ERS-220[A] model AIBO, with working WiFi card OR
  • An ERS-7 model AIBO (with built-in WiFi card)

  • A Pink Programmable Memory Stick (AKA "PMS" or RCode stick) - ERA-MS008 or ERA-MS016, $34->$39
  • A memory stick reader to install the software (Use only Sony brand stick readers)
  • A Windows PC with a WiFi card on the same network as the AIBO

NOTE: if you have an original ERS-210 [not A] and have never run an OpenR SDK program before, you may need to perform the Flash Update on your ERS-210.

Installation:

  • Put an available PMS stick into your computer's stick reader
  • Select any folders you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. Do not format the stick.
  • Download the proper version of AiboSpeedRacer for the model of your AIBO.
    • For the ERS-2x0: AiboSpeedRacer2x0.zip (1.5MB, ERS-210, ERS-220, ERS-210A or ERS-220A)
    • For the ERS-7: AiboSpeedRacer7_ERS7M3.zip (1.6MB). NOTE: do not install on the AiboMind stick, use a PMS (a stick without an ERF- number)
  • Unzip this RCodePlus file to the root of your PMS memory stick.
  • Copy your working WLANCONF.TXT file for your LAN setup to the \OPEN-R\SYSTEM\CONF folder on the memory stick. Otherwise the default settings will be used (AIBONET, WEP enabled "AIBO2", ip address 10.0.1.100).
    NOTE: the WLANCONF.TXT file is in a different folder for RCode/RCodePlus/LifePlus sticks.

  • Download the Windows Speed Racer Control Program: RacerControl.zip (92KB)
  • Unzip that file to a folder on your PC. The program is called 'RacerControl.EXE'

Usage

  • For the ERS-7, be sure the WiFi switch is "ON". For the ERS-2x0, be sure the WiFi card is in and working.
  • If you have a custom WLANCONF.TXT, be sure you know what your AIBO's IP address (default: 10.0.1.100)
  • If you are using an ad-hoc WiFi connection, be sure your PC WiFi card is active (with link LED on or blinking)
  • Put the memory stick inside your AIBO and start it up.
  • Aibo will boot and stretch into a crouching position with head down and knees bent.

  • Run the RacerControl.EXE program on the PC
  • Enter the IP address for your AIBO and Connect.
  • The main display should appear and the camera video should show up on your PC's screen. AIBO's head should raise up.
    If you have connection problems, be sure you can 'ping' your AIBO from the PC.

  • When connected, use the buttons to make AIBO walk:
    • "Fwd" for forward, "Stop" for stop, "Bwd" for backward
    • "Fwd L" and "Fwd R" will walk on an angle
    • "Side L" and "Side R" will walk sideways
    • "Turn L" and "Turn R" will make AIBO turn in place
    • You can set the walking speed with from "Slow" to "Hyper". Change the speed first, then press one of the Walk direction buttons
    • Optionally use the Kick buttons to kick the ball with the dog's head or body. These are more aggressive than the walking and may jam your AIBO.
      NOTE: the very aggressive WalkBench body slam moves are not supported by RacerControl.

  • Camera Settings: You can use the 9 "Camera" buttons in the upper right to adjust the camera settings. The 3x3 buttons are the same as the AiboRemote and AiboScope programs. If the screen is dark, press the "H" and "S" buttons to increase the camera gain and slow the shutter speed.

  • NOTE: if AIBO jams, please reboot your AIBO (and reconnect the RacerControl program)
  • When done, remove the memory stick from AIBO (this will shut him down). Do not press the pause button.
