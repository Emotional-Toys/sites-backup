PleoRB Sensor Overview

The sensor values are not backwardly compatible with the original PLEO firmware. You must recompile your PDK program to make it compatible with the RB. For use with YAPT, you must have an updated .Y file.

    RFID values

    Each RFID toy has a code number. Each has its own PCB with the number written on it (if you want to rip apart the toys be careful). Here are the known toys and their numbers.

      The first 8 cards are used to train the voice recognition commands

    • 01 - the ID card used for naming the PleoRB (comes with PleoRB). All ID tags appear to be the same (ie. the serial number printed on top is cosmetic).
    • 02 - Come To Me stone (vr_come_to_me)
    • 03 - Counting stone (vr_stay_with_me)
    • 04 - Playing stone (vr_lets_play)
    • 05 - Eating stone (vr_eat_now)
    • 06 - 'Bow' stone (comes with PleoRB, vr_go_to_sleep)
    • 07 - Singing stone (vr_sing_a_song)
    • 08 - Dancing stone (vr_dance)

      Tug-of-war is a special game mode (vr_tug_of_war)

    • 09 - Tug of War stone on rope (comes with PleoRB)

      The others

    • 12 - Conifer Leaf (Health Care)
    • 13 - Cycad Leaf (Daily Nutrition)
    • 14 - Sugar Cane (Leisure Food)
    • 15 - Chili Pepper (Leisure Food)
    • 16 ($10) - Coffee Leaf (Leisure Food)
    • 17 ($11) - Mushroom (Daily Nutrition)
    • 18 ($12) - Mint Leaf (Health Care)
    • 19 ($13) - Rock Salt (Health Care)
    • 20 ($14) - Chunk of Ice (Daily Nutrition)

      There are some unavailable 'cheat' cards (exp_cheat_card)

    • 10 - Cheat Card #10 - reset all properties to zero, except DNA; start over as newborn (age_new=10)
    • 11 - Cheat Card #11 - advance/regress to hatchling (age_new=20)
    • 21 ($15) - Cheat Card #21 - advance/regress to social development (age_new=30)
    • 22 ($16) - Cheat Card #22 - advance to juvenile (age_new=40).
    • 23 ($17) - Cheat Card #23 - advance to mature/adult (age_new=50). Set mood properties to low values (9)
    • 24 ($18) - Cheat Card #24 - advance to mature/adult (age_new=50). Set mood properties to midrange values (50)
    • 25 ($19) - Cheat Card #25 - advance to mature/adult (age_new=50). Set mood properties to high values (99)
    • others TBD
    • 124 ($7C) - something used internally for voice recording?
    Note: the default BabeRB personality will speak the RFID code when it detects the toy.
    Note: the soft rubber leaf does not have an RFID chip inside.

    Note: the 'Cheat Cards' are inferred by looking at the robot software. AFAIK they are not sold as true toys.
