RoboBoa IR Codes

The RoboBoa modulation scheme is the same as the RSV2 12-bit encoding described at the top of the IR Codes V2 page. The first 4 bits are "0111" for the RoboBoa (hex $7).

Direction Pad

  • $700 = ?

  • $701 = top (up) = head right
  • $702 = bottom (down) = head left
  • $703 = left = neck right
  • $704 = right = neck left
  • $705 = top+left = upper right
  • $706 = top+right = upper left
  • $707 = bottom+left = down right
  • $708 = bottom+right = down left
  • $709 = pad middle = stop (twice to reset to default position and aware)

  • $70B = Track = track moving things
  • $70C = Explore = explore the room
  • $70D = Alarm = set alarm (repeat to add another hour)
  • $70E = Guard = enter guard mode (10 minutes)
  • $70F = Demo = demo mode
  • $710 = Execute = Program Play

  • $712 = mode button alone (hold for 2 sec)

    Mode shifts (hold mode button down, press button within 2 sec)

  • $715 = MODE+top = robot lifts and travels forward for 1 min
  • $716 = MODE+bottom = robot lifts and travels backward for 1 min
  • $717 = MODE+left = robot lifts and spins clockwise for 1 min
  • $718 = MODE+right = robot lifts and spins counter-clockwise for 1 min
  • $719 = MODE+top+left = robot moves to light up the ceiling on left
  • $71A = MODE+top+right = robot moves to light up the ceiling on right
  • $71B = MODE+bottom+left = robot moves to light up the floor on left
  • $71C = MODE+bottom+right = robot moves to light up the floor on right
  • $71D = MODE+middle = shutdown

  • $71E = MODE+Tail/Head = Strafe
  • $71F = MODE+Track = Lazer
  • $720 = MODE+Explore = Cannon
  • $721 = MODE+Alarm =???
  • $722 = MODE+Guard = Patrol (10 minutes)
  • $723 = MODE+Demo = cycle volume control
  • $724 = MODE+Execute = Back (undo last step when in program mode)

  • $725 = MODE button alone (press and release within 2 secs) = mode light (remote controlled spotlight)

    With tail/head shift on (red LED on)

  • $729 = TAILHEAD+top = body up
  • $72A = TAILHEAD+bottom = body down
  • $72B = TAILHEAD+left = tail backwards (wheel)
  • $72C = TAILHEAD+right = tail forward (wheel)
  • $72D = TAILHEAD+top+left ...
  • $72E = TAILHEAD+top+right ...
  • $72F = TAILHEAD+bottom+left ...
  • $730 = TAILHEAD+bottom+right ...
    NOTE: TAILHEAD+middle will send regular hold ($709)

  • $732 = HOLD Tail/Head for 2 sec = Reset (and clear all settings)
  • $733 = HOLD Track for 2 sec = Scan
  • $734 = HOLD Explore for 2 sec = Sleep
  • $735 = HOLD Alarm for 2 sec = Daily Alarm function
  • $736 = HOLD Guard for 2 sec = Sentry mode (10 minutes)
  • $737 = HOLD Demo for 2 sec = Party mode
  • $738 = HOLD Execute for 2 sec = enter program mode

  • $??? = HOLD middle for 2 sec = hold (desklamp mode)
  • $??? = HOLD other direction pad??
  • $??? = MODE+button for 2 sec???
  • $??? = HOLD MODE+Alarm = test alarm function (typo?)
