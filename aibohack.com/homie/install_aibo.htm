HomieGate Install - AIBO

The current AIBO version is very simple. AIBO will sit there and wait for you to tell it what to do. A future version will be more elaborate and inco

Hardware Requirements

  • HomieGate Server installed, configured and tested previously

  • An ERS-210 model AIBO OR
  • An ERS-220 model AIBO

  • A Pink Programmable Memory Stick (AKA "PMS" or RCode stick) - ERA-MS008 or ERA-MS016, $34->$39
  • A memory stick reader to install the software (Use only Sony brand stick readers)
  • An AIBO LAN Card on the same WiFi network as the HomieGate Server
You should be familiar and comfortable with general AIBO WiFi configuration and testing (using 'ping'). If you have never run "AiboScope LIVE!" or "AiboRemote" or the Sony AIBO Navigator, you should start with those programs before trying HomieGate. If you have run those programs before, and have a working WLANCONF.TXT file, you are most of the way there.

Installation and Testing

Installation (on AIBO)

  • Put an available PMS stick into your computer's stick reader
  • Select any folders you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. Do not format the stick.
  • Download the proper RCodePlus runtime for the model of your AIBO.
    • For the ERS-210: RCode Plus 2.52 'A' (210)
    • For the ERS-220: RCode Plus 2.52 'A' (220)
  • Unzip this RCodePlus file to the root of your PMS memory stick.
  • Download the HomieGate Test Personality: HG_Aibo01.zip(very small 35KB)
  • Unzip that file on top of your RCODE/PMS stick (replacing any files it asks about).

  • If necessary, customize the WLANCONF.TXT file on the memory stick to work with your WiFi LAN setup. If you don't change this, the default will be for a WiFi 'AIBONET' with WEP enabled ('AIBO2' password). AIBO will be on IP address 10.0.1.100
  • Put the stick in AIBO (but don't boot him up yet)

Installation (on HomieGate Server)

  • Run the configuration tool (C:\Homie\hg_config.exe)
  • Check the "AIBO (Client #3)" checkbox.
  • Change the IP address for your AIBO if necessary (NOTE: be sure you are running .01'A')
  • Press the "Save Changes and Generate Voice Commands" button

Testing - LAN connection and polling

  • Be sure your WiFi card is running on the PC (usually the Link light will flash)
  • If the HomieGate server is running on the PC, stop it now.
  • Put the memory stick inside your AIBO and start it up.
  • AIBO will stretch and sit there waiting with no LEDs on.
  • Now run the HomieGate server (C:\Homie\hg_server.exe).
  • Keep an eye on the third status line in the upper right of the HomieG server dialog. It should change from "Searching for AIBO" through several messages eventually to "AIBO ready"
  • AIBO should stretch again, and his forehead light will turn green.
  • TIP: If AIBO is not found, be sure you can 'ping' the AIBO from your PC
  • TIP: if you can ping, but have problems, try closing and restarting the HomieGate Server (hg_server.exe on the PC).

Testing - Voice Commands

  • Before saying any HomieGate voice command, say "AIBO" first (or press his back button). AIBO's lights will start flashing.
  • When the lights are flashing, say the phrase (eg: "How old are you?")
  • If AIBO/HomieGate doesn't understand, it will beep. Please say "AIBO" again, and then say the phrase again.
  • TIP: using the back button is necessay in loud rooms, or if AIBO gets confused. Press the back button gently so it doesn't make any noise.
  • TIP: if you asked AIBO to play music, it will be too loud for him to hear you. Press his back to give him a new command (like "Stop the Music"). He will mute the stereo to listen to your every word.

That's it for setup. Try some of the long list of what to say (voice commands).

Other Tips

  • It usually helps to stop the HomieGate Server program (hg_server.exe) when you initially boot up your AIBO.
  • Some of the canned AIBO skits are a little 'jerky', but should not harm your AIBO
  • Multiple AIBOs are supported. The need to be on separate IP addresses and configured properly with hg_config.exe. The configuration tool supports two AIBOs, but the server can handle up to four (the architecture can handle even more).
  • With multiple AIBOs, try not to talk to them both at the same time. If both are in the same room, use the back switch to talk to one at a time.
