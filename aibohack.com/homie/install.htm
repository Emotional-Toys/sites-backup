HomieGate Install - Main

This is the main installation steps for the HomieGate server. HomieGate is recommended for experienced computer people. Some of the installation steps are rather complicated. LAN (WiFi) setup can be very complicated.
Advice: There are many things that can be done with HomieGate. You don't have to get them all working in one evening. If you run into problems, relax and take a break. Computers can be frustrating at times.

Hardware Requirements

  • Windows PC (the 'Homie Gate Server PC')
  • WiFi card (or Access Point) attached to the Server PC. WiFi components should be setup and working already!
  • Microphone on the Server PC (for testing).
  • Speaker on the Server PC (for testing and for regular use to play music)
  • Windows XP is recommended.
    Earlier Windows versions will work as well.
    Very old Windows versions (like Win98) are not recommended for use as a full featured HomieGate server (with Voice Recognition)
  • The faster the processor the better. Voice recognition is more responsive with a 2GHz+ CPU, but a 500MHz CPU also works acceptably.
  • Disk space requirements for HomieGate are minimal. Large hard disk(s) may be used for your music collection.

Installing Voice Recognition

First check if you need it

You will need a SAPI 5.x compatible speech recognition and text-to-speech engine installed on your computer. You can check to see if this already installed by running the 'Control Panel' and selecting 'Speech'. The 'Speech Properties' Panel should have both a 'Speech Recognition' and 'Text to Speech' tab.
If one of these tabs is missing, or you don't have a 'Speech' control panel, then you need to install the Speech SDK. If you have both, you can skip the next step.

Download and Install the Speech SDK

The Speech SDK version 5.1 is available for free download from Microsoft. This is a large download (68MB).
NOTE: do not use the newer beta 6.x SDK.
NOTE: you will probably have difficulties on Windows 98 or Windows ME. Windows XP (or Windows 2000) are recommended for the voice recognition features.

Steps:

  • Download the "Speech SDK 5.1" : speechsdk51.exe (68 MB) That is not a typo, it is 68 Mega-Bytes in size.
  • When downloaded, run the 'speechsdk51.exe' program, and specify a folder (like c:\VSETUP) to unzip the files to.
  • Run the SETUP.EXE program in that folder
  • Accept the licence agreement (read it first if you prefer)
  • Install the SDK in a different folder (like C:\VSDK)

When done you can delete the C:\VSETUP folder and its contents.
Optional you can train the speech recognition engine to better understand your voice. I recommend you do that later.

Installing HomieGate

NOTE: you must install HomieGate in the 'C:\Homie' folder on your PC to start with. After the initial installation and testing, you can move this elsewhere if desired. You must edit paths in the HG.INI file.

Steps:

  • Download HG_Preview01b.zip (2MB, with config fix and music.g fix)
  • Unzip it to the "C:\Homie" folder on your Server PC. Do not install it in another folder.

(what the folder should look like at the start)

Configuring HomieGate (the first time)

Before running the server, you need to configure it.
  • Run the configuration tool (C:\Homie\hg_config.exe).
  • Press the "Find My Music" button. This will find the two sample albums ("total 15 tracks from 2 albums". Later you can incorporate your real music collection.
  • Press the "Save Changes and Generate Voice Commands" button at the bottom. HG_config.exe will update the HomieGate server information and close.

There are more features of the hg_config.exe program which you can use later.

Testing HomieGate

For this step we will be using the same PC for both the server and the client.
After this initial test, you will use other remote devices ("clients") connected using WiFi. This includes PDAs, SD panels or robots like AIBO.

NOTE: you will need a microphone attached to the PC to say voice commands. The default "Sound Recording" device is used. The volume of your microphone must be set correctly. This is the recording volume (not the system speech recognition volume).

Run the Homie Client

  • Run the PC version of the client, (C:\Homie\hg_client.exe)
  • It will ask for an IP address. We are using the localhost 127.0.0.1, so you don't have to change anything.
  • Press "Connect"
  • It should start with the standard client interface in a small window.

    Please perform the following tests in the following order:

    Test simple .MP3 music playback

  • Expand the tree view under "Music"/"Pop"/"Madonna - Like a Prayer".
  • Pick any of the .MP3 track and it should play on the PC's speakers. Note: these are short low quality clips, not the full album.
  • If that doesn't work, you may not have a .MP3 decoder setup.
    Try the four Cartman test .WAV files under "Music"/"TestWavs". If WAVs play but not MP3s, please check the     Music setup FAQ.

    Test microphone and memo record

  • Press the "Memo" button
  • Say something into the microphone on your PC (talk for a few seconds)
  • When done, press the "Done" button (formerly the "Memo" button)
  • Scroll down to the "Voice Memos" list and pick the one on the bottom (the files are time stamped in a format: 'yyyy.mm.dd hhmm')
  • When you pick that voice memo, you should hear a recording of what you recorded played back on your PC's speaker
  • If you don't hear it, your microphone level may be set incorrectly, or your PC sound card may be confused. Try playing back "Cartmans Todo Note". If you hear it, then check the microphone volume. If you don't hear it there may be something wrong with your sound card setup.

    Test voice recognition (the cool features)

  • Press the big "Push to Talk" button.
  • Ask "What time is it?". Speak clearly into the microphone.
  • You should hear the answer, and the big button should change back to "Push to Talk".
  • Repeat the process (ie. press the button before you talk) and ask:
  • Ask "Why is the sky blue?"
  • Ask "What is the first law?"
  • Ask "Who invented the internet?"
  • Say (in one breath) "Play Music Like a Prayer by Madonna"
  • Say "Next song"
  • Say "Stop the music"
  • There are literally thousands of things you can say or ask. See the What to Say list.
  • NOTE: When HomieGate is listening to you, the "Push to Talk" button becomes "Talk Now, push to stop". Normally you don't need to press the button to stop Normally if HomieGate understands what you are saying, you don't have to push the button a second time to stop.

    What Now ?

  • If you have an X10 Firecracker module controlling your home, you may want to try the X10 setup now. Otherwise, you should start installing the client software for WiFi devices you own.
  • If you have an AIBO with a WiFi card, you will probably be eager to install the AIBO support
  • If you have a hacked SD panel, you should perform the SD Panel install since it is the best remote control around.
  • PocketPC users should look into the PDA install if you have the right hardware.
  • Later, you should think about incorporating your music collection sitting on your computer (perhaps in MP3 format). See the music music setup page.

    If you don't have any of the above, you can stick with the PC based client, but you are really missing out on the coolness of HomieGate.
