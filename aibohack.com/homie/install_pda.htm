HomieGate Install - PDA Install

The PDA version is great for advancing play lists or selecting one or two songs or turning on an X10 device. The current version is the same as the general WinCE version. It is not yet optimized for the PocketPC user interface.
Voice recognition is not recommended. Please try recording a voice memo. On my e740 the microphone is too noisy to use for voice recognition. If you have better luck, please let me know.

Hardware Requirements

  • HomieGate Server installed, configured and tested
  • An ARM based, WiFi PDA like the iPaq or Toshiba e7xx line.
  • The PDA WiFi Card must be on the same WiFi network as the HomieGate Server PC

Installation and Testing

Installation

  • Copy the program "C:\Homie\PDA\hg_client_arm.exe" to the PDA (typically in the "\Windows\Start Menu" folder)
  • Run the 'hg_client_arm' program on your PDA
  • Enter the IP address of the server (this is the only time you will need the virtual keyboard). You may have problems getting the keyboard to popup. If you have problems you can try to have the keyboard up before starting hg_client_arm for the first time.
  • Press "Connect"

Usage

  • You should know what to do by now by using the other PC clients.
  • If you can get reliable voice recognition with the PDA microphone, please let me know.
