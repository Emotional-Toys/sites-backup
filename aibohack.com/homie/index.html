HomieGate Beta 3

HomieGate is a program that runs on your PC (Windows 98 and later) that provides a gateway to a remote device connected via WiFi.
The next release is BETA 3. It has not yet been released. The rest of this documentation is for background and future (work-in-progress) installation instructions. BETA 3 that adds CLIE PDA support, AIBO ERS-7 support and a few other things. The old ERS-210/220 protocol (from BrainBo and HomieGate Beta1) is no longer supported. Please wait for Beta 4 if you want to use the ERS-210 or ERS-220 with HomieGate.
New server features include support for additional built-in features (Sony CX777 400Disc CD/DVD player, MIDI music players). Also planned is a Girder plug-in for almost any other computer or home automation needs. NOTE: The optional Girder is not free, but is pretty cheap ($20).

HomieGate

  • Overview - What is 'HomieGate' ?

    Installation:

    • Base install and testing (required)
    • Optional Voice Fonts (optional)

    • ERS-7 RFW AIBO install (optional)
    • ERS-210/220 RFW AIBO install (not yet supported)
    • Palm/CLIE PDA install (optional)
    • SD Panel install (optional)
    • PocketPC PDA install (optional)

    • Music install (your music collection, optional)
    • Firecracker X10 install (home automation, optional)
    • Girder install (home automation, others, optional)
    • Other features (piano, CX777; optional)

    Usage:

    • What to say: General voice commands
    • What to say: ERS-7 AIBO specific voice commands
    • What to say: AliceBot smarts

    • Music: Playing tracks, albums, playLists

    What's new in HomieGate BETA 3

    • Palm client, for PalmOS PDA with WiFi (full voice features for CLIE UX/TH or older NX/NZ/TG running MCA2). Also works with WiFi enabled PalmOne T3.
    • VoiceFont support (optional, if you have purchased voice fonts separately)
    • Totally revamped Aibo polling using RFW for the ERS-7
    • Lots of cool ERS-7 specific features (like training AIBO to memorize new objects using visual pattern recognition)
    • Midi support (download thousands of MIDI tracks from the Classical Musical Archives and easily integrate into your home stereo or MIDI instrument).
    • Minor update to AliceBot-like knowledgebase (including "Who is the Governor of California?")
    • Built-in support for the Sony CX777ES (400 Disc CD/DVD player)
    • ??? Girder plugin for almost anything else

    • ERS-210/220 using an RFW-like approach to polling (but won't have the really cool features of the ERS-7 - no pattern recognition)
