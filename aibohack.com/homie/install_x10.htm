HomieGate and X10 commands

DISCLAIMER: Currently only the FireCracker X10 PC control module is supported.
If you have another X10 computer interface module, please send me an email with details.
HomieGate will control X10 devices by voice commands or by selecting options from a HomieGate client (in the tree view under 'X10 Devices').
NOTE: HomieGate is not a replacement of your existing home automation software, it only provides basic on/off control on demand (ie. no timers, no macros, no automatically turning lights on and off).

Hardware Requirements

  • HomieGate Server installed, configured and tested
  • An X10 "FireCracker" module (CM17A) connected to a serial port on the PC
  • An X10 Transciever module (TM751) set to the house code you want to use (eg: "A"). This also functions as a lamp module (eg: "A1")
  • One or more X10 devices in the same house (eg: lamp modules, appliance modules, garage door openers, ...)
    NOTE: All of these components come in a FireCracker starter kit for around $40 USD for people in the USA.
    NOTE: In future the more powerful Active Home (CM11A/CM12) will be supported.

Configuration

  • Run the configuration program (C:\homie\hg_config.exe)
  • Check the "Use X10 (FireCracker)" checkbox
  • Set "Com Port" to the COM/Serial port number your FireCracker is attached to
    NOTE: the Firecracker module has a pass-through, so you can connect other serial devices to the COM port, but they must not be used while HomieGate is running.

  • Press the "Edit" button. This will edit the file 'hgdata_x10.txt' (usually in notepad).
  • Make any changes to the hgdata_x10.txt file. Save the file and exit the text editor.
  • Return to hg_config.exe and press "Save Changes and Generate Voice Commands". You must do this after every change to 'hgdata_x10.txt' so the voice commands will be available.

  • If the server is already running exit it (close box or ESCAPE)
  • Start the server now (C:\homie\hg_server.exe) and it will use the new settings.

  • Run one of the HomieGate client programs, like the PC or SD Panel or a PDA, and connect to the HomieGate server.
  • Test out your new additions by selecting the devices in the tree view under 'X10 Devices'
  • Try out the voice commands as well.

Editing hgdata_x10.txt

The format is very simple, and described at the top of the file itself. You can easily add an unlimited number of X10 commands to your HomieGate server by editing X10.TXT
