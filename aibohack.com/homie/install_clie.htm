HomieGate Install - CLIE Install

The CLIE version is great for advancing play lists or selecting one or two songs or turning on an X10 device.
Voice recognition does work, and you can use your CLIE to

Hardware Requirements

  • HomieGate Server installed, configured and tested
  • Palm OS 5.2 CLIE PDA with built-in WiFi - UX50 or TH55
  • or Palm OS 5.0 CLIE PDA with external WiFi card and MCA2 installed - NX60, NX70, NZ90, NX73 or NX80.
  • The CLIE WiFi Card must be on the same WiFi network as the HomieGate Server PC

Installation and Testing

Installation

  • Install the program "C:\Homie\PDA\HomieClient.prc" to the CLIE (typically with Hotsync)
  • Run the "HomieClient" program on your CLIE
  • It will startup the WiFi connection
  • Enter the IP address of the server. Leave the Password field blank.
  • Press the OK button and it will try to connect.
  • If it fails to connect, check your WiFi connection to the PC server. Try using the Prefs panel to 'ping' the server.
  • See the advanced login page for more advanced features

Usage


  • You should know what to do by now by using the other PC clients. The user interface is generally the same, but there are differences. The tree-view list is similar to the Windows tree control. You will need to use the stylus for scrolling through long lists of items, or expanding/contracting nodes.
  • The little triangles in front of items act like the "+" and "-" boxes under Windows. Press a triangle or an item with a triangle in front of it and it will expand or contract.
  • Pressing an item without a triangle will activate that item (eg: play a Music track, an entire album, activate an X10 device etc)

  • The jog dial, page up and page down buttons will navigate through the tree-view list.
  • The Back button starts the voice command function (alternatively the 'T' key). Press the back button, and a "Talk Now" alert will pop up. You have 5 seconds to talk. Give a HomieGate voice command like, "What time is it?".
