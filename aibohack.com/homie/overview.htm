HomieGate Overview

Architectural Overview

The main HomieGate server program runs on your desktop PC (the Homie Gate Server). It is the gateway to your Home PC.
Some available features require hardware attached to the Homie Gate Server (eg: Home stereo, X10 home automation).

Remote devices (clients) talk to the Homie Gate Server via WiFi. You use these devices to control the features of the Homie Gate Server, either through a touch screen or through voice commands.
Examples of HomieGate voice commands include "what time is it?", "turn on the light", "play album Ray-of-light by Madonna", "Skip the song", "why is the sky blue?".

For use with AIBO, everything is done through voice commands. You get AIBO's attention, by saying "AIBO" followed by the HomieGate voice command. ??? For use with PDAs, there is a simple user interface that you can click directly on the PDA screen. Everyting is directly accessible, ie. no typing (other than entering the initial IP address of the server). This is much more convenient for picking songs from albums to play, or building quick play lists, or direct control over X10 devices.

The voice recognition feature is also available on PDAs (when the PDA microphone is adequate quality). You can also record voice memos and other features.

Supported Clients (Remote Devices)


The following clients will be supported for the Beta3 release:

AIBO

  • - ERS-7 AIBO (with built-in WiFi card).

    • Speak your commands into AIBO's ears. He will talk back to you and do other things (ie. dance or walk around).
    This is similar to the old BrainBo V1. Older version of HomieGate (Beta1) supported the ERS-210 and ERS-220. The ERS-2x0 models are not currently supported in the current release.

    CLIE

    Supports the CLIE PalmOS devices with WiFi support (eg: NX/NZ with WL1x0 card, UX or TH series). Voice recognition and synthesized voice playback supported (NOTE: requires MCA2 for NX/NZ devices).

    Viewsonic Smart Display Panel

    Use microphone and speaker. Large screen is great for selecting from large music collections.
    Same as the PocketPC version.

    Pocket PC

    Any Pocket PC PDA with ARM processor and WiFi card (eg: Toshiba or iPaq). Small PDA version of user interface. All the same features as the large panel display. Depending on device, microphone may be too noisy for good quality voice recognition.

    Local Client (Windows)

    The client software will run on a standard Windows PC as well. This is typically run on the server PC to test out the interface, but can be used on any other PC (eg: a laptop) with a WiFi card.

    Server Features

    The following PC features/peripherals will be supported in the preview release:
  • Knowledge base (based on AliceBot and BrainBo V1), eg "Why is the sky blue"
  • PC local stereo device (use with free home stereo server, high quality optional)
  • Simple voice memos
  • Optional: X10 home automation control using FireCracker interface (CM17A only) for now
  • Optional: CX777 support, Sony's 400 CD/DVD/SACD player. Select disc or track from tree view.
  • Optional: Piano support, Midi output to sythesizer or other MIDI music player. Data from "Classical Music Archives" can be easily integrated.
