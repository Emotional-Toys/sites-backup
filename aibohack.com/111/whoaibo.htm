[AiboHack] Who AIBO 11x

Who AIBO 11x

"Who AIBO" is for Doctor Who fans who always wanted their AIBO to talk like K9. Be nice to your AIBO. If he gets mad, he will turn into a Dalek.
WhoAibo says many Dr Who themed phrases, many of which include the word "exterminate".

Supported AIBOs

  • - ERS-110/111 is supported.
  • - ERS-2x0 and 31x supported with teamdax's version of Who AIBO 2x0/31x

Installation

  • Put an available 8MB or 16MB purple stick into your computer's stick reader
  • Select any folders or files you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. (if you see a file named MEMSTICK.IND, you can leave it)
  • Download the YART_11x basic personality: RCode11x with YART.zip (Save this .ZIP file on your computer. You may want to use it again later.)
  • Unzip this file to the root of your memory stick.
  • Download the WhoAibo personality: whoaibo10.zip
  • Unzip that file on top of your memory stick (replacing any files it asks about).
  • Put the memory stick inside your AIBO and start it up. Boot time is a little long (so enjoy the repeating intro WAV). It will start as a nice "K9" AIBO.
  • Play with your AIBO. Follow the instructions (see below).

    Usage

    WhoAibo has four moods: Nice K9, Active K9, Nice Dalek, Bad Dalek/Davros.
    WhoAibo starts out as Nice K9. The more you hit your AIBO (hard press on the head), the more his personality will turn negative. The more you pat your AIBO (soft press on the head), the more his personality will turn positive. You can also directly control WhoAibo's mood with the remote.

    Remote Control

    With the ERS-11x sound controller, you can control WhoAibo just like the regular Yart_11x personalities (eg: Game mode to move him around, play back performances etc).
    With WhoAibo, the performance styles also set the current mood.
    • 31+Send - MSTYLE1 - Set Nice K9 style/mood
    • 32+Send - MSTYLE2 - Set Active K9 style/mood (he will walk around)
    • 33+Send - MSTYLE3 - Set Nice Dalek style/mood (he will be your friend)
    • 34+Send - MSTYLE4 - Set Bad K9 style/mood (he will try to exterminate you or rant like Davros)
    Leave WhoAibo sitting around in each of the four moods to hear the full extent of his personality.

    Customization

    You can customize the Who AIBO personality using YART11x. To run YART_11x, just open the "\CUSTOM" folder on the memory stick and run the YART_11x.EXE program in there.
    You should be familiar with YART_11x and have run through the tutorial first. WhoAibo's personality is more complex (four moods) than the standard RCodeWithYart personality (which has just one mood).

    Disclaimer: WhoAibo is a good "performance" personality - ie. it will say about 70 interesting K9 and Dalek themed things. It is not a great "autonomous" personality to use in your AIBO all the time (like the normal AiboWare that came with the dog). If you want to improve the logic (especially needed in the area of walking around and exploring and ball playing), please feel free to do so (the R-CODE source is in the \OPEN-R folder on the stick).
