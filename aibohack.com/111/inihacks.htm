INI file hacks

This file contains information on customizing you AIBO using some undocumented configuration files. These files do not exist on the AiboWare memory stick, so the AiboWare software uses defaults for these values.
To use them, just create a file with the proper name in the "/open-r/app/conf" directory of the memory stick. With the exception of the "[DEBUG_GROWING]" line, all other lines in these files should be a string followed by a space followed by a number. Don't add extra blank lines.

BNTDEBUG.CFG

 /open-r/app/conf/bntdebug.cfg sets debugging options for the BeNet (Behavior Network). Here's the default for main options. 
GROWING 1
LEARNING 1
EMOTION 1
INSTINCT 1
You can disable features by changing the value to "0".

If you disable the emotional model (EMOTION=0) or the instinctive model (INSTINCT=0), then you can also specify a fixed emotional and/or instinctive state: 
anger ###
affection ###
appetite ###
curiosity ###
disgust ###
exercise ###
fear ###
joy ###
sadness ###
surprise ###
Where ### is a number from 0 to 100.

There are additional options for printing debugging info. I believe they go to the serial port. 
EMO_PRINT 0
INS_PRINT 0
SEM_PRINT 0
DST_PRINT 0
COM_PRINT 0
PRINT_DATA 0

TOUCH.CFG

 The /open-r/app/conf/touch.cfg file lets you customize the default thresholds for the dog's head sensor. 
TOUCH_TIME 10
THRESHOULD_TIME 37
THRESHOULD_TIME_LONG 500
These times are in 1/100ths of a second and decide the difference between CLICK HIT, PAT and LONGPAT.

There is also a "CALC_MAX" feature, that is normally on: 
CALC_MAX 1
Turning it off affects how CLICK is handled in some unknown way.

There are also values for pressure settings, but I'm not sure how you enter them in the TOUCH.CFG file. The default values are: 
MIN_PRESSURE .1
THRESHOULD_PRESS .3

LION

 /open-r/app/conf/LION sets options for AiboWare's learning module.
The contents of the file (with ERS-111 defaults) looks like: 
[DEBUG_GROWING]
BABY 100000
CHILD 300000
YOUNG 1000000
ADULT 4000000
You can adjust the thresholds to any value you want. Values are in 1/10 second of "quality time".
