"Peter Pan Syndrome"

Peter Pan Syndrome

Peter Pan Syndrome (PPS) is an affliction of AIBO hardware that prevents the AIBO personality from properly evolving.
If you have never tested your AIBO for "PPS" then you should do it at least once.

Detecting PPS

There are three ways of testing for PPS.

Using Test AIBO!

 This test runs on your AIBO. You will need a memory stick reader and a spare memory stick. Email me for Test AIBO 11x.

Using AIBO Browser (automatic way)

 This test runs on your PC. You will need a memory stick reader to analyze your dog's memory stick.
For this test, you need the AIBO Browser version 2.1 or later. When you open a memory stick with the AIBO Browser, it will alert you if it detects possible PPS.

Using AIBO Browser (manual way)

 You can use the AIBO Browser to manually track your dog's progress. If the dog's "Quality Time" stops going up, then you dog may have caught PPS.

Strains of PPS

There are several causes, usually related to flash memory, but all end up with the inability of AIBO to save it's brain state to it's memory stick when the Pause button is pressed.
I've broken them down into two strains based on which flash memory is afflicted.

Wassup! Strain PPS

There are several ways that the internal Flash ROM memory can get corrupted. Most of them will be corrected by the AiboWare.
If corrupted in a certain way, the internal Flash ROM memory will prevent the AiboWare from updating data properly.
If you have used the old (version 1) Wassup AIBO! memory stick in your AIBO, this may have triggered this strain of PPS in your dog.

Detection:
Any of the three PPS tests should detect this strain. Replacing the memory stick software does not fix the problem.
Cure:
One drastic fix is to replace the body hardware of the AIBO.
A much easier fix is to use the "Deckard Fix" (see below) which will correct for most Flash memory corruption errors.

Toshi Strain PPS - AKA Toshi's Syndrome

A lesser know strain of "PPS" is known as "Toshi's Syndrome". The cause of this is a low level format problem of the memory stick itself. There is nothing wrong with the AIBO, just the memory stick.

Detection:
Test AIBO! (if you use the problematic memory stick) will detect this strain.
Using the AIBO Browser and keeping an eye on the Quality Time is the only sure way of detecting this case.
Replacing the memory stick software does not fix the problem.
Cure:
Reformatting the memory stick usually fixes this problem.
Worst case, you may need to replace the memory stick.
Don't forget to reinstall AiboWare onto the memory stick.

If you suspect a different strain of "PPS", please email me.
