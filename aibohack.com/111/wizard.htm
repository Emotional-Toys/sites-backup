AWizard - the AIBO Performance Data Wizard (for the ERS-11x)

The AIBO Performance Data Wizard can:
* extract performance data files from your AIBO's memory stick
* combine performance data files and write them to your AIBO's memory stick

There are three different editing modes supported:

Mode 0: works like the AIBO Performer Kit

The user performances are extracted from the memory stick (MOTION.ODA and AUDIO.ODA). The performance data is saved on your computer in the specified directory. The performance data is renamed so that Audio and Motion data are combined in user friendly action names.
There are 86 performances you can edit in this mode.

Mode 1: raw user performances

Not recommended, but included for completeness. Exact same data as Mode 0, but without renaming. You must figure out which audio files go with which motion files.
