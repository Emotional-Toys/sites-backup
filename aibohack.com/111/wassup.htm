Wassup AIBO!

Wassup AIBO! is a cheap knock-off of Hello AIBO! (ERA-101), a more expressive and active AIBO personality.

Installing Wassup AIBO

  • You will need an 8MB or 16MB purple memory stick.
  • Backup your memory stick if appropriate.
  • Delete all the existing files on the stick
  • Download this file wassup2.zip (2.7MB) and unzip it on top of the blank stick
  • When complete, put the stick into your AIBO and start him up
  • After playing, you can restore your backup, or leave the personality on the memory stick

Interacting with Wassup AIBO

Interact like you would normally. Wassup AIBO! includes "Super Game Mode" (see below).

Super Game Mode - Instructions

The Super Game Mode is an extension to Game Mode. Be sure to install Wassup AIBO! first.

Because of the peculiarities of the Sound Controller, getting the dog into Super Game Mode is a little tricky.

When the game controller is in game mode, the four LEDs will be flashing.
When the dog is in game mode, the dog's eyes will flash green.

    Step1) Get the dog into game mode. Press 01+Send (do not press the game button on the controller yet). The dog's eyes will flash green and will do a bow when ready.
    Step2) Decide which style you want the dog to use (see below). Press the proper sound command, for example 44+Send (for trot mode). The dog will do another bow.
    Step3) Now press the "GAME" button on the controller once. The 4 LEDs should be flashing green now.
    Step4) Press the direction keys on the keypad to move the dog around. The dog will continue in the direction that you send it. Hit the "5" key to stop the dog.
To change modes, you can press the "GAME" button one more time and the controller and the dog will leave game modes. Alternatively, you can press the "ON" button on the controller (4 LEDs no longer flashing) and start over at Step2 above.

NOTE: do not press the "TRACK" button on the controller while in Super Game Mode otherwise the custom walk style will be lost.

To set the walking style send one of the following commands to the dog in game mode.
    31+Send: Limping
    32+Send: Baby Walk
    33+Send: Slow Walk
    34+Send: Crawl
    41+Send: Medium walk
    42+Send: Funny walk (AKA Crab walk)
    43+Send: Normal walk
    44+Send: Trot
    45+Send: Fast walk
