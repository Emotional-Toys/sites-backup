Star Trek AIBO

StarTrek AIBO by Genie.


Live AIBO and Prosper

(artwork also by Genie)

Supported AIBOs

  • - ERS-110/111 is supported.

Installation

  • Put an available 8MB or 16MB purple stick into your computer's stick reader
  • Select any folders or files you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. (if you see a file named MEMSTICK.IND, you can leave it)
  • Download the YART_11x basic personality: RCode11x with YART.zip (Save this .ZIP file on your computer. You may want to use it again later.)
  • Unzip this file to the root of your memory stick.
  • Download the StarTrek Aibo personality: startrek10.zip
  • Unzip that file on top of your memory stick (replacing any files it asks about).
  • Put the memory stick inside your AIBO and start it up. Boot time is a little long (so enjoy the repeating intro WAV).
  • Play with your AIBO. You should be able to figure that out by yourself.
