Virtual AIBO - 3D AIBO



Overview

The main tool for viewing and editing AIBO performances is Virtual AIBO. Virtual AIBO is a 3D model of the AIBO that is used for playback and editing of motions.
For convenience, Virtual AIBO runs in a separate "top-most" window that can be positioned anywhere you want. You may close or minimize the window when Virtual AIBO is not needed.

Moving Around Virtual AIBO

You can use your mouse to grab the 3D picture of the dog. If you use the left mouse button it will rotate the dog. Hold down the SHIFT button to rotate the dog in a different axis.
If you select the Virtual AIBO window, you can use the NumPad+ (ADD) key to move AIBO closer, and the NumPad- (SUBTRACT) key to move further away. You can resize the Virtual AIBO window to be as small or large as you like.

Virual AIBO and Playback

Virtual AIBO is used to Playback AIBO motions in the Browser (ABrowser.EXE) and Trainer (ATrainer.EXE) tools.
In these modes you can only move the entire AIBO. When some performances are played back, some of AIBO's limbs may be grayed out. This indicates the performances do not use those limbs.

Virual AIBO and Editing

Virtual AIBO is used for both Playback and Editing of AIBO motions in the Editor (AEditor.EXE). Virtual AIBO is attached to the current selection in the track editor. When you select or change something in the track editor, Virtual AIBO will show you what it looks like. You can select the different parts of the dog and rotate specific limbs.
Just for fun, you can rotate the dog's ears and toes (nothing is saved).

Platforms

Virtual AIBO is implemented in the VIRTAIBO.DLL. It requires DirectX (version 5 or better) for the 3D Virtual AIBO.
The file VIRTAIBO.DLL must be in the same directory as the main program.
