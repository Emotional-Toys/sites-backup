YAPT = YAPT advanced topics (OBSOLETE INSTRUCTIONS)

NOTE: before trying these advanced topics, please try the standard YAPT tutorial with the Babe personality.

YAPT can be used with pre-existing personalities like Holiday Pleo (Xmas songs) or PleoPM (the default personality) that weren't meant to work with YAPT. The old logic is replaced completely. You can reuse any of the resources (sounds, motions or animation 'commands').

Changing Holiday Pleo

The Holiday Pleo personality randomly picks between 6 Christmas songs. For this tutorial, we are going to use YAPT to make a predictable behavior removing the random playing. Pressing one of 6 sensors (head, back, 4 legs) will perform one specific song. This is useful if you have more than 1 Pleo and want to trigger all of them to perform the same song.
  • Prerequisite: Please work through the intro YAPT tutorial first, with the Babe personality.
  • Prerequisite: Please download and run the Holiday Pleo program first
  • If you haven't installed it already, download the Holiday Pleo from the UGOBE site and install it on the SD card. There should be a PLEO.URF file in the top-most folder of the SD card.
  • Download the yapt01a.zip stand-alone install and unzip it to the top-most folder of the SD card. (there should be 2 .AMX files, 2 .EXE files, one PLEO.Y file and the PLEO.URF file)

  • Run the YAPT.EXE program.
  • Initially, there is no behavior. Select "Startup" and press the top "Add" button. Add the motion "x_hohohoa". That's a short intro skit so we know the Pleo has booted up properly.
  • At the main YAPT screen, select "Body"/"Head Release". Add the motion "x_song_merry_xmas".
  • At the main YAPT screen, select "Body"/"Back Release". Add the motion "x_song_jingle2".
  • At the main YAPT screen, select "Legs"/"Left Leg Release". Add the motion "x_song_12_days".
  • At the main YAPT screen, select "Legs"/"Right Leg Release". Add the motion "x_jingle_bells".
  • At the main YAPT screen, select "Legs"/"Left Arm Release". Add the motion "x_song_goodking".
  • At the main YAPT screen, select "Legs"/"Right Arm Release". Add the motion "x_deckhalls".
  • Geek Note: in the original Holiday Pleo, the random song selection is done with an Animation Command "x_rxn_song". You can assign it to another sensor if you like. It will randomly pick a song like the original.

  • When you are done, press the "Save program to SD card" button on the bottom. Then press "OK".
  • If necessary 'stop' the SD card (required for some computers) and remove the SD card from the SD card reader and insert in Pleo SD slot.
  • Test it on Pleo. Pleo should say "ho ho ho" when it starts.
  • To trigger a song, hold one of the 6 assigned sensors (Head, Back, or 4 legs). Hold your hand on the sensor for a second or two. When you release your hand the song will start.
  • NOTE: The leg sensors are on the upper part of the legs (not to be confused with the foot sensors on the bottom of the feet)
  • NOTE: songs are not interruptable (that's how the YAPT personality works for now), so you will be forced to listen to the entire song ;->

  • NOTE: For synchronizing multiple Pleos, release the sensors all at the same time.
  • ADVICE: For quick reactions the 'Tap' sensors should be used. Pleo will react instantly. For longer reactions the 'Hold' sensor condition should be used. This reduces the chance of a mis-trigger (eg: picking up Pleo). Finally, use the 'Release' sensor condition if you want to be in control of the exact start of a skit (for making videos or multiple Pleo performances)

  • You can repeat the YAPT process with other motions and sounds on the Holiday Pleo personality. However there aren't that many canned motions to pick from.

Changing PleoPM (the default personality)

Short answer: don't.
Use the YAPT approach with the 'Babe' personality or a MySkit created personality. These personalities have a sound and motion library at the end that is very similar to the PleoPM default personality.

Browsing PleoPM (the default personality)

You can use YAPT (and P3D) to browse the built-in sounds/motions/commands of the default personality (PleoPM), or any PLEO.URF you find.
  • "Grab" the default PleoPM personality from your Pleo, to an SD card. Copy the Pleo.URF file to a folder on your PC. Instructions for Grabbing PleoPM are here. You only need to do that once.
    The rest can be done on your PC (no SD card or Pleo required).

  • Download the yapt01a.zip stand-alone install and unzip it to that folder containing the PLEO.URF file.
  • Run YAPT. For this purpose, most of the features of YAPT are ignored. Press the top "Add" button and YAPT can be used as a sound/motion/command browser.
  • In the "Pick Action to add" dialog, check the "Auto Preview" checkbox at the bottom right.
  • You can browse through the Sounds, Motions and Commands of the default personality. When you select them, they will be previewed in the P3D player.
  • For browsing purposes, don't double-click or "Add Action". When you are done with browsing, you can close YAPT and ignore any changes.
