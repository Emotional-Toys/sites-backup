CliePet's CLIE page

Generally useful programs/hacks

  • FMP - Full Movie Player - full screen movie player (landscape - UX/VZ, portrait - TH/NX/NZ) - Version .03

  • MCA2 - Modern-Clie-Audio System Extension provides standard audio support for NX, NZ and TG model CLIEs.
  • AlbumSwap utility for quickly switching albums/playlists with the built-in CLIE AudioPlayer.
  • SpyStillCam - use your CLIE's camera to transmit 320x240 photos to your PC via WiFi. New version 2.1 with JPEG compression (most CLIEs with WiFi and camera)
  • BouncyCam - fun little bouncy camera for the UX/NX/NZ/TH. BouncyCam.prc (now with menus)
NOTE: some programs are model specific. If your CLIE model number does not match, please don't bother trying them, they won't work.

For developers and advanced users

  • VZ90 owners only: VZ90 English - partial translation for system features
  • VZ90 owners only: RotVZ Explicit screen rotation control for CLIE VZ90 (only!)

  • ClieMounter utility - for enable/disable of MS or CF volume (NX/NZ tool). ClieMounter.prc.
  • Time_RawVFS - run to test raw i/o speed of Memory Stick or Compact Flash. Time_RawVFS.prc
  • Demo App: MyCam - simple camera demo app MyCam.prc (now with menus)

  • HomieGate supports WiFi enabled CLIEs (or T3+WiFi card) in the upcoming Beta 3 release. See the main HomieGate page for an introduction to HomieGate. Email me if interested in trying out (works best for the home music player control)
