CLIE SpyStillCam (2.1)


Version 2.1 - with JPEG compression A simple CLIE SpyCam and PC based SpyCamWatcher program. Supports UX, NZ, NX(V) model CLIEs. This is similar to the old version 0.2 SpyCam. It is called SpyStillCam since the data is transfered from the CLIE to the PC as small JPG still images. Frame rate will vary depending on the speed of the CLIE and polling rate.

A Windows based PC with WiFi card is required. Transmits JPEG compressed video. You can get several frames per second by adjusting the sampling rate and lowering the compression quality.
You can change the camera settings from the PC (and run the CLIE unattended).

Hardware Requirements

  • CLIE with the built in camera (NX-V, NZ, UX)
  • Power cable for the CLIE is recommended. It can run from battery (and will not auto-shutoff until the batteries get very low)
  • CLIE WiFi card (built in for the UX, WL100/110 card for the NX/NZ)

  • A Windows PC
  • A WiFi card for your PC
  • NOTE: Your CLIE WiFi and PC WiFi must be on the same network (ie. so you can 'ping' your CLIE from the PC)

Installation

  • Download this file SpyStillCam21.zip to your computer and unzip it.
  • Install the SpyStillCam.prc on your CLIE.
  • Put the SpyCamWatcher.exe program somewhere on your PC.

Running It

  • If running in 'ad-hoc' mode, be sure your PC's WiFi card is running (link light blinking first)
  • Run SpyStillCam on the CLIE. It should connect and show you the preview window.
  • SpyStillCam also shows a guess of the CLIE's IP address at the top of the screen. This is just a guess. To be sure run the "View Log" option from the Network Prefs panel.
  • The CLIE is now a spy cam. You should plug it into a charger and hide it somewhere. Running from battery for an extended period of time is not recommended.

  • Run the SpyCamWatcher.exe program on the PC.
  • Enter CLIE's IP address (usually the "Possible IP address" displayed on the CLIE itself) Press "Connect".
  • If you get connection problems, check to see if you can 'ping' your CLIE using that IP address.
  • Assuming there no connection problems -- watch the video!
  • The default starts with 1 frame per second. Press the "Auto fast" button to run the video at full speed.
  • You can also adjust the CLIE camera settings remotely from your PC.
  • The Quality setting adjusts the quality of the JPEG compression. Lower quality means smaller image size, and faster frame rates.
  • NOTE: when exiting the app, you may have to press on the main launcher icon for 1 second before it registers.
Settings names are based on the UX/NX7x camera (eg: White Balance names and Effect names). They are slightly different for the NZ90. For the NZ90 the preview window shows the reticule and flash status which is somewhat annoying.

NOTES: uses TCP/IP port 51009. If you are using a fire-wall or similar program, please be sure to enable that port.
