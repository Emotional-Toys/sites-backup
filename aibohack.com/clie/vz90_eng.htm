VZ90 English Patches

For English users who imported the Japanese VZ90, here are some helpers to customize some of the trickier parts of the system: (NOTE: please don't bother to download unless you own a VZ90)

Installation

  • Check on the back of your PDA and look for the model number "PEG-VZ90". If you can't find it stop right now.
  • Backup your PDA (eg: using MS Backup). If you don't like the english mods, it is easier to hard reset and restore the backup.
  • You must already have FileZ installed on your PDA, and have a general idea of how to use it.
  • If you have previously installed an earlier version, delete any 'ovly' or 'panl' types in RAM. Soft reset.

  • On your PC
    • Download vz90_eng_01.zip
    • Unzip it to a memory stick on your PC (eg: place in unique folder like "ENG_VZ90")
  • Place memory stick into VZ90
  • Run FileZ on the VZ90
    • Select all the .prc files from the memory stick (eg: from "ENG_VZ90" folder)
    • Copy to the files to the "Internal" memory
  • Soft reset the device

Testing it out - Prefs

After the soft reset, you will begin in Prefs. Notice it is mostly in English now, with a smattering of Japanese.
The General prefs panel lets you pick the color scheme, including the new cool "Black" scheme.
You can ignore the "UserDict" prefs panel (not needed for English users).

Testing it out - Launcher

By default the Japanese Media Launcher will be the main screen. You can press the small 3x3 graphic in the upper right to get to the program launcher.
To make the program launcher the default, in any of the Media Launcher apps (Movies, Music, Photos) look for the second menubar with a "Media Launcher ???..." item. Pick that menu item, and un-check the checkbox.

If you haven't done so already, you can edit the program categories ("Edit Categories...") and change all the initial Japanese names to something in English. NOTE: there is one hard coded string for the 'Unfiled' category that will remain in Japanese (ie. the one right before "Edit Categories...").

NOTE: if you use a different launcher program, you don't need to install the Launcher.prc file to RAM (saves ~200KB of RAM).
NOTE: "Buttons", "Extension" and "UserDict" will appear in the program list (for deleting and copying). Irritating.

Testing it out - Other

Other system utility apps like Calculator, CardInfo, Security and HotSync are mostly in English.

Other Tips

Now you should have a mostly functional CLIE PDA for the core system services. The only other thing to remember is if the Grafitti input area has a small button on the lower right - press it to switch to ASCII mode if needed.
For other programs, the easiest way is to find an English program that does what you want and install it in RAM. See my "Day 3" report for other advice.

To get rid of programs you don't need, you can use SharkROM. I like hiding "ATOKPrep", "CLIE Organizer", "ClieFAQ", "JEDict" and "Welcome".
