Disco AIBO (for the ERS-11x, 2x0, 31x, and ERS-7)

Installation instructions are different for the different model AIBOs. Please follow the correct instructions

Supported AIBOs

  • - ERS-111 is supported
  • - ERS-210 is supported
  • - ERS-220 is supported
  • - ERS-311/312 is supported
  • - ERS-7 is supported

Overview

Disco AIBO makes your AIBOs dance along to songs played on your computer.
  • Disco AIBO works with all model AIBOs (ERS-11x, ERS-2x0, ERS-31x and ERS-7)
  • Disco AIBO has a total of 22 dances
  • All dances have ERS-7, ERS-31x, ERS-2x0, ERS-11x parts
  • Some dances have multiple parts (ie. singer and backup dancers) for the ERS-2x0 and ERS-31x

    The tones at the start of the songs provided trigger AIBOs to start dancing at (approximately) the same time.

    Disco AIBO is built using RCODE/RCodePlus. For the ERS-2x0, ERS-31X and ERS-7 versions, you will need a "blank" PMS/RCODE stick. The ERS-11x portion can be used on any memory stick.

    WARNING: the first time you playback any Disco AIBO steps, be sure your AIBO is on a soft surface (carpet recommended). Some dance steps may cause AIBO to fall down. That is part of the charm of the dances. Help out a fallen AIBO be helping him stand/sit up and continuing to dance.
    As always use at your own risk.

    Installation for ERS-7 with PMS stick


    NOTE: The old AiboMind1 game version has been discontinued. You will need a PMS stick.

    Installation:

    • Put an available PMS stick into your computer's stick reader. You can't use the 32MB AiboMind pink stick.
    • Select any folders you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. Do not format the stick.
    • Download the RCODE7+YART memory stick image: RCODE7_YART703_E.zip (3.8MB)
    • Unzip the memory stick image on top of your RCODE/PMS stick.

    • Download this additional file disco7pms.zip.
    • Unzip it to the root of the memory stick. It will ask you to replace some files. Answer "Yes".

    • Put the memory stick inside your AIBO and start it up.
    • Play with your AIBO. Follow the instructions (see below).

    Installation for ERS-210, 220 or 31x


    Required Hardware:

    • An ERS-210 model AIBO OR
    • An ERS-220 model AIBO OR
    • An ERS-31x model AIBO (311 Latte, 312 Macaron or 31L Pug)

    • A Pink Programmable Memory Stick (AKA "PMS" or RCode stick) - ERA-MS008 or ERA-MS016, $34->$39
    • A memory stick reader to install the software (Use only Sony brand stick readers)

    Installation:

    • Put an available PMS stick into your computer's stick reader
    • Select any folders you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. Do not format the stick.
    • Download the proper RCodePlus runtime for the model of your AIBO.
      • For the ERS-210: RCode Plus 2.52 'A' (210)
      • For the ERS-220: RCode Plus 2.52 'A' (220)
      • For the ERS-31x: RCode Plus 2.52 'A' (31x)
      • (Save this .ZIP file on your computer. You may want to use it again later.)
    • Unzip this RCodePlus file to the root of your PMS memory stick.
    • Download the personality for Disco AIBO: disco45(all).zip (~450KB)
    • Unzip that file on top of your RCODE/PMS stick (replacing any files it asks about).

    • Put the memory stick inside your AIBO and start it up.
    • Play with your AIBO. Follow the instructions (see below).

    Installation for ERS-110, 111


    This covers installing Disco AIBO on an ERS-110 or ERS-111. Disco AIBO will work on any 8MB or 16MB memory stick. A purple memory stick is recommended (save the red sticks for the ERS-210).
    • You will need an 8MB or 16MB purple memory stick.
    • If you are reusing an existing stick, backup first. You can restore your backup later.
    • Delete all files from the memory stick
    • Download this file disco45(11x).zip (2.6MB) and unzip it on top of the existing stick.
    • When complete, put the stick into your AIBO and start him up
    • Play one of the songs (below) on your computer. Make sure your AIBO can hear the first part of the song. AIBO will dance along with the song.
    • If AIBO falls on his face, help him up (some dances are not tuned for the 11x)
    • After playing, you can restore your backup, or leave the Disco personality on the memory stick
    • Shut down AIBO when done, or just pull out the memory stick.

    Using Disco Aibo, The Songs

    Using:

    AIBO should be sitting down ready to start when you play a dance song. NOTE: Disco AIBO doesn't do anything else.

    Play one of the songs (below) on your computer.
    AIBO must hear the first part of the song. AIBO will dance along with the song.
    If AIBO falls on his face, help him up. If AIBO jams, unjam it (by holding the head) and it will restart the personality and wait for the next dance to begin. When done with dancing, shut down AIBO by pressing the chest button (or just pull out the stick).

    Multi-part dances

    Two part AIBO dances require at least two AIBOs. Set one to be the lead-singer part. Set the second AIBO(s) in the backup-dancer part. Then play the song as usual.

    Three part AIBO dances require at least three AIBOs. Set the first one to the lead-singer part. Set the second one to the second-lead part. Leave the third (or any other) AIBOs in the backup-dancer part. Then play the song as usual.

    To change the part played by an ERS-2x0, press his back sensor. It will cycle through the parts. The right-red eye indicates the "lead-singer" part. The left-red eye indicates the "second-lead" part. The green eye (the default) indicate the "backup-dancer" part.
    Press the head to show the current mode.

    To change the part played by an ERS-31x, press his tail up direction. It will cycle through the parts.
    The orange/amber dome light indicates the "lead-singer" part. The blue dome light indicates the "second-lead" part. The green dome light (the default) indicate the "backup-dancer" part. Press the head to show the current mode.

    The ERS-7 and ERS-11x versions have a single part.

    Special Features

    To abort the current dance:
  • ERS-2x0: Hold down the back sensor for 3 seconds
  • ERS-31x: Hold the tail in the up direction for 3 seconds
  • ERS-7 PMS: Hold the top and bottom back sensors for 3 seconds.

    The personality will restart and wait for the next dance to begin.
