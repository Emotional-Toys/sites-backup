Toshiba e800/e805 UnDead Hack V2.3 (longer instructions)

NOTE: For the original Toshiba e800/e805. Do not use if you have upgraded to Pocket PC 2003 "SE".

The "Dead zone" is a feature / bug of the Toshiba e800/e805 PDA that prevents the stylus from working in the upper left corner of the PDA when in VGA mode. This is normally where the Start Menu is located (in normal portrait orientation).
Installing the UnDead Hack will correct this problem.

Requirements

You must have the following:
  • Toshiba e800 or e805 PDA
  • ROM version 1.0108 or 1.0110. Other versions may work.
  • An ActiveSync connection for installation
  • Installed one of the following resolution switchers. You should be familiar with how the VGA <-> QVGA switching process works.
    • ResFix (1.0) by deez, free.
    • ResSwitch (1.9) by envoke, free.
    • ResFix (2.0) with landscape mode by deez, $10.
    • MyVGA by hugbug and others, free but registration required
    • NOTE: it will also work with the built-in Toshiba ClearVue limited VGA mode, but that is a lame experience.

  • NOTE: if you have installed an older version of the UnDead Hack, the new installer program will help you uninstall it (potentially saving 3/4 MB of RAM)

Installation

This is a one-time installation (ie. do it once and forget about it, unless you hard reset your PDA). You can optionally keep the "install_undead.exe" program on your PDA for later use. It also serves as an un-install program.
  • BACKUP YOUR PDA FIRST!
  • If you have any programs that you plan on disabling or uninstalling later, temporarily disable or uninstall them now. Anything program installed (and active) before the undead fix is installed will become fixed in the chain of handlers. In these more complicated cases, do not try to uninstall those older drivers or the undead fix. Well behaved programs you install after the undead fix can be uninstalled later.
  • Some common examples:
    • NYDITOT driver - disable NYDITOT before running install_undead.exe (Special Case #5)
    • Calligrapher, is not well behaved. Install it before the undead hack. Do not try to uninstall later. This is Special Case #4 and you must confirm.
    • Well behaved input programs (eg: Effy and other IME programs) can be installed before or after the undead hack.
    • If you install a well behaved before the undead hack, it will ask you to confirm (Special Case #4), after which you should not try to uninstall.
    • If you install a well behaved after the undead hack, there should be no problems and you can uninstall the program later (but leave the undead hack installed).

  • Download this file: install_undead.exe [V2.3 13.8 KB], and put it somewhere you will remember (eg: on your PC desktop)
  • NOTE: Do not run this program on your PC. It is a Pocket PC program that you must run on your PDA.

  • Put your PDA in the cradle connected to your PC.
  • Perform the following steps on your PC:
    • Microsoft ActiveSync should popup and connect [if not check your connections]
    • Press the "Explore" button.
    • Double click on "My Pocket PC"
    • Double click on the "Temp" folder
    • Drag the install_undead.exe program from your PC (eg: your desktop) to the \Temp folder on the PDA

  • Perform the following steps on your PDA:
    • Run "File Explorer" (in Programs) and go to the "\Temp" folder
    • Click on the "install_undead" program
    • If successful, it should report "UnDead hack installed OK (version 2.3)" and "Soft reset to activate". If not, check below for "Special Case #?"

  • Remove the PDA from the cradle, and press the reset button. After the soft reset the patched driver will be used.
  • Enjoy - the dead zone is dead!

Special Case#1 - uninstall of version 1

If the install_undead program detects the old version 1 patch (patched GWES.EXE)
  • it will report "Patched GWES.EXE found" and will ask you if you it should move the file for deleting later.
  • Answer "YES".
  • After rebooting, go back to the "Temp" folder on the PDA and you will see "GWES_DeleteMe.exe". Delete it and you will free up 3/4MB of RAM.

Special Case#2 - keep program or delete it

You don't need to keep the install_undead.exe program around after following the main instructions and rebooting. If you plan on uninstalling it later, you can keep the program around (perhaps moving it to the "\Windows\Start Menu\Programs" folder). Otherwise you can delete it from the "\Temp" folder.

Special Case#3 - uninstall

If you run the install_undead.exe program on a PDA that already has the UnDead hack installed, it will ask you if it should uninstall the hack. However, do so at your own risk. This will restore the "dead zone" and depending on the current screen resolution, this may cause problems for you.

To upgrade, you need to run the newer "install_undead.exe" twice. First uninstall the old version, soft reset and then install the new version (and soft reset again). NOTE: during this process, the dead zone will be back temporarily. Switch to QVGA mode first if you want to be extra careful.

Special Case#4 - programs that chain the touch.dll driver

Some programs like NYDITOT, Effy and Calligrapher will take over the touch driver chain. This is a non-standard use of the WinCE system registry, and unfortunately most programs do it differently.

If there is something already installed in the chain when install_undead.exe is run, it will report "Warning: another touch driver is installed", and ask "Do you want to install on top of it?" You will normally select "YES", and it will add the undead hack to the current head of the chain. This is a safe option, as long as you don't try to uninstall programs installed before the undead hack.

NOTE: It will also report the DLL name that is currently at the head of the chain (eg: "\Windows\cgrtouchstub1.dll" for Calligrapher 6.x, "Effy_Touch_??.dll" for Effy, and others possible). This is mostly just for-your-information.
NOTE: If you find chaining programs that do not work after you select "YES" in the install, please send me an email.

Special Case#5 - NYDITOT

A special case of Special Case#4. Disable the NYDITOT driver, pick the "(Disable Virtual Display)" mode and soft reset. Then try the install (install_undead.exe) again.

Other problems that don't have a "Special Case#?" number are probably general errors. Try resetting your PDA and trying again.
