Toshiba e800/e805 UnDead Hack V2.3 and PadRot 1.2

NOTE: For the original Toshiba e800/e805. Do not use if you have upgraded to Pocket PC 2003 "SE".


Getting rid of the "Dead Zone"

Undead Version 2.3 - now works with Italian ROMs (and hopefully other international ROMs) and fixes a bug introduced in 2.2 with Calligrapher
[note: if version 2.1 is working for you, there is no compelling need to upgrade]
PadRot 1.2 - fixes jog dial for more compatibility with Internet Explorer
The standard disclaimer applies: Use at your own risk!
Be sure to perform a full backup of your PDA first, just to be safe.

Full Install Instructions

Recommended for first time installation:
  • UnDead Hack - Removes 'dead-zone' when in VGA mode
  • PadRot Hack - Rotates D-Pad buttons and scroll wheel orientation when in landscape or upside-down mode

Quick Install Downloads/Instructions

If you know what you are doing: (Toshiba e800/e805 only)
  • BACKUP YOUR PDA FIRST!
  • Download the files and transfer them to your PDA
    UnDead Hack: install_undead.exe [V2.3 13.8 KB]
    PadRot Hack: install_padrot.exe [V1.2 13.8 KB]

    NOTE: Do not run these program on your PC. They are Pocket PC program that you must run on your PDA to install, uninstall or re-install the patches.
  • Run the program once to install the patch
  • Run the program again to and it will ask if you want to un-install the patch
  • To re-install, run the program once to uninstall the old patch, reboot and then run the program again to install the new patch
  • If you have problems, especially with Calligrapher, please refer to the longer "Full Install" instructions.
