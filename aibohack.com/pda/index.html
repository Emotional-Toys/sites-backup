Aibo Related Tools for Windows PDAs

AiboCam - WinCE/Pocket PC 2002 (WiFi required)

ARM processor only (including 'StrongARM' and 'XScale').

NOTE: This is a generic WinCE program. It will have an old style menu bars on a PocketPC (ie. on the top) rather than the PocketPC style along the bottom. It will run on larger screen PDA devices running Windows CE (including the Viewsonic Smart Display Panel).

Installation / Usage

  • Download wpda_aibocam_beta.zip and unzip it to your PC.
  • Copy it to your PDA. Typically using the "Explore" feature of ActiveSync, place the AiboCam.exe program in the "\Windows\Start Menu" folder for easy access.
  • Be sure your PDA's WiFi card is enabled and setup to the default AIBO LAN.
  • Start up the PDA, making sure the link light is blinking (if applicable).
  • Boot up AIBO.
  • Run the AiboCam program. Enter AIBO's IP address if different from the default (NOTE: non-standard IP addresses not currently saved)
  • It should connect within a few seconds. If not it will take about a minute to time out if AIBO is not found. If AIBO is not found, check the general advice above, and in particular be sure you can 'ping' both the PDA and AIBO.
  • DISCLAIMER: if the LAN settings are not setup just right it can be a frustrating experience!
  • When connected, you can pick the camera update speed from the menu (or Manual polling).
  • BETA NOTE: this version will report the frame count. If a transmission error is detected, it can take a minute to clear out (and the video will freeze). If this happens a lot, let me know.

General WiFi recommendations

  • An ERS-210[A] or ERS-220[A] AIBO with AIBO LAN (WiFi) card is required
  • Your AIBO must be running RCodePlus or LifePlus and be properly configured for the WiFi LAN.
  • Be sure to run the PC version of AiboRemote (for RCodePlus) or AiboScope (for LifePlus) _before_ trying the PDA version
  • The PDA's WiFi card must be setup to use the standard AIBO LAN network (by default: AIBONET, with WEP key "AIBO2")
  • The default AIBO IP address (10.0.1.100) is recommended (unless you can't use it in your setting)
  • Be sure you WiFi card's "Link" light is flashing before you boot your AIBO!
  • Use of an access point is recommended to keep the WiFi connection alive, especially during the boot process.
  • During initial setup, a PC with WiFi card (for ping tests) is recommended.
  • If you can't connect, be sure you can ping both the AIBO (usually 10.0.1.100) and the PDA from your PC.
