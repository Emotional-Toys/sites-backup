Toshiba e800/e805 PadRot V1.2

NOTE: For the original Toshiba e800/e805. Do not use if you have upgraded to Pocket PC 2003 "SE".

Normally when your PDA is in regular portrait orientation, the D-Pad buttons will move the cursor around properly. If the screen is rotated to landscape (left or right handed) or upside-down portrait mode, the D-Pad buttons will not work as expected.
Installing the PadRot Hack will correct this problem.

Features:
  • Adjusts direction pad buttons to work properly with the current screen rotation
  • Version 1.2: keeps the jog dial acting like page up/down (not left/right)
  • Version 1.2: proper DPAD/ROCKER notifications when rotated
  • Optionally: Adjusts app launch buttons to rotate
  • Single install, all other adjustments automatic

Requirements

You must have the following:
  • Toshiba e800 or e805 PDA
  • ROM version 1.0108 or 1.0110. Other versions may work.
  • An ActiveSync connection for installation
  • Previously installed one of the following resolution switchers. You should be familiar with how the VGA <-> QVGA switching process works.
    • ResFix (2.0) with landscape mode by deez, $10.
    • MyVGA by hugbug and others, free but registration required
    • NOTE: PadRot is not needed for regular Portrait mode VGA

Installation

This is a one-time installation (ie. do it once and forget about it, unless you hard reset your PDA). You can optionally keep the "install_padrot.exe" program on your PDA for later use. It also serves as an un-install program.
  • BACKUP YOUR PDA FIRST!
  • Download this file: install_padrot.exe [V1.2 13.8 KB], and put it somewhere you will remember (eg: on your PC desktop)
  • NOTE: Do not run this program on your PC. It is a Pocket PC program that you must run on your PDA.

  • Put your PDA in the cradle connected to your PC.
  • Perform the following steps on your PC:
    • Microsoft ActiveSync should popup and connect [if not check your connections]
    • Press the "Explore" button.
    • Double click on "My Pocket PC"
    • Double click on the "Temp" folder
    • Drag the install_padrot.exe program from your PC (eg: your desktop) to the \Temp folder on the PDA

  • Perform the following steps on your PDA:
    • Run "File Explorer" (in Programs) and go to the "\Temp" folder
    • Click on the "install_padrot" program
    • If successful, it should report "PadRot hack installed OK (version 1.2)" and "Soft reset to activate". If not, check below for "Special Case #?"
    • Optionally you can close that dialog and it will then ask, "Rotate App Launch buttons?". Normally you should answer no. This is the default.

  • Remove the PDA from the cradle, and press the reset button. After the soft reset the patched driver will be used.
  • The easiest way to test this is to run the Pocket Excel program. Use the Direction pad to scroll around the 2D array of cells. When the screen is rotated (with ResFix2 or MyVGA) the Direction pad buttons will scroll in the correct direction.

(there is no Special Case#1)

Special Case#2 - keep program or delete it

You don't need to keep the install_padrot.exe program around after following the main instructions and rebooting. If you plan on uninstalling it later, you can keep the program around (perhaps moving it to the "\Windows\Start Menu\Programs" folder). Otherwise you can delete it from the "\Temp" folder.

Special Case#3 - uninstall

If you run the install_padrot.exe program on a PDA that already has the PadRot hack installed, it will ask you if it should uninstall the hack.

To upgrade, you need to run the newer "install_padrot.exe" twice. First uninstall the old version, soft reset and then install the new version (and soft reset again).

Special Case#4 - programs that chain the keyboard driver

If there is something already installed in the chain when install_padrot.exe is run, it will report "Warning: another keyboard driver is installed", and ask "Do you want to install on top of it?" You will normally select "YES", and it will add the padrot hack to the current head of the chain. This is a safe option, as long as you don't try to uninstall programs installed before the padrot hack.

NOTE: It will also report the DLL name that is currently at the head of the chain
