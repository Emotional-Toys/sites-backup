    Overview - what is the map editor ?

    The map editor is a Windows program that edits the gamesave of the Nintendo DS game "Animal Crossing:Wild World" (or the Japanese version "Oideyo Dobutsu no Mori").

    If you have a Homebrew NDS you can use my Trainer NDS app or a backup tool like ETool or RAIN. If you have an Action Replay you may be able to update it or modify it to work with the AC:WW game Click Here for more details..

    With the map editor, you can customize your inventory, or add additional money ('Bells'). You can customize the world map, placing rocks or trees wherever you want. Also you can leave items laying around to pickup later. You can customize the rooms of your house in traditional and not-so-traditional ways.


    USE AT YOUR OWN RISK.
    NOTE: Keep an original backup of your game save in a safe place. If you go crazy with the editor you can always go back to the backup layout.

    Hardware Requirements

    • NDS game "Animal Crossing:Wild World" (or the Japanese version "Oideyo Dobutsu no Mori"). I will use "Animal Crossing" to refer to either.

      • If you have a Homebrew NDS, you can run a backup tool (like ETool or RAIN) or the NDS Homebrew trainer. Click Here for more info on the Homebrew trainer

    • A Windows PC to run the map editor program (AnimalMap.EXE).

    How to use it with Action Replay DUO (not the newer "Action Replay DS")

    If you have an Action Replay, follow these instructions:
    NOTE: be sure you have already tested the AC:WW game with your action replay, done any necessary modifications, and made a backup copy of your gamesave.
    • Get a current backup of the game (if you don't have one already in your "PC Vault") ClickHere for a reminder of the steps.
    • Using the Action Replay software. Decompress it to a .DUC file (eg: "test1.duc"). ClickHere for a reminder of the steps.
    • I recommend renaming saving a backup copy of the file. That's the last known working version. You may want to use it later.

    • Double click on the Map Editor program (AnimalMap.EXE) to run it.
    • Select a game save to open. This must be a valid Animal Crossing game save (English or Japanese version). For Action Replay open the .DUC file (eg: "test1.duc")
    • Perform the desired editing operations described below.
    • When done, press the "FINAL SAVE" button. It will replace the file with the changes you have made (in this case "test1.duc")

    • Using the Action Replay software. Import the new DUC file to your PC Vault. Click Here for a reminder of the steps.
    • Using the Action Replay software. Restore the new gamesave to the cartridge. Click Here for a reminder of the steps.
    • When complete, turn off the NDS console. Remove the Action Replay unit.
    • Remove the AC:WW game cartridge from the Action Replay unit. Stick it in the top of the NDS console like normal. Start up the NDS console.
    • Play the game as you would normally. Changes you have made to your Inventory, cash balance ('Bells'), world map or other things should appear in the game.
    • If the game fails to boot, you may have made too extravagent changes (hard to do in basic editing mode, easy to do in the advanced modes). Go back to an early backup and try again.

    FAQ/TIPS:

    If you get an error message "ERROR: bad checksum #1" or "ERROR: bad checksum #2" or "ERROR: mirror copies don't match", don't panic. Nothing is wrong, the editor is being extra safe and won't let you change the game save in this state.
    There is a simple work around. It takes a little longer during the initial backup - but not as long as re-running the Action Replay download process.
    Before doing the AR backup step, quit the game and save ("Save and end.") the first time, as you would normally. Then start the game again ("Continue" from the title screen, pick the same player). Wait until the town is prepared ("Preparations Complete") and you walk out the front door. Immediately quit the game and save ("Save and end.") a second time.
    Then do the Action Replay backup process.

    Action Replay can be flakey. "The console is not connected" message is very common. Follow the advice in the AR manual, and the information on this website, including the AR Help page.

    The AR socket may not fit snuggly. Try re-inserting the AR unit into the NDS slot or jiggle it. You should be able to get it to boot in regular mode (don't hold the L/Left shoulder button) and backup the DS cartridge - space permitting.

    If your display is 1024x768 (minimum recommended screen size), the larger dialogs may not fit on the screen. One workaround is to turn off your taskbar or make it autohide (give enough room for the very bottom). OR you can grab the title bar and drag the entire window up (ie. so the title bar is reduced to 1 pixel at the top of the screen). That should give you enough room to see the very bottom of the dialog.

    How to use it - Homebrew style

    If you have a Homebrew NDS, you do essentially the same thing as the above. Replace the backup and restore steps with running a gamesave backup tool like ETool, RAIN or my trainer. Once uncompressed, the map editor will be able to edit the gamesaves. The compressed format for my trainer is supported directly.
    The testing process of updating the gamesave is usually much faster than the Action Replay.
