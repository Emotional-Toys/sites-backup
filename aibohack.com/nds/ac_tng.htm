This is a teaser page to show you what is possible. For details on how to do it, please goto this page.

There are two different "Action Replay" devices. The older "AR MAX/DUO" which does gamesaves, and works with the map editor ($35, getting hard to find). The newer "AR DS", which does 'codes' ($25).
I don't recommend you get the "AR DS" alone. If you only buy one, buy the "AR MAX/DUO". However, if you want really cool AC:WW hacks, buy both and you can do some clever tricks, without risking your own town.
