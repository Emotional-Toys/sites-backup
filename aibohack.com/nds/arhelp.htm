Action Replay Helper Topics

See the Main Action Replay page for overview, installation, updates and mods. This page contains short summaries of the common Action Replay options.

Make a backup from the AC:WW game cartridge to your PC

  1. Insert the AC:WW game into the "DS" slot of the Action Replay
  2. Insert a GBA game in the "GBA" slot of the Action Replay (any GBA game, it won't work without it)
  3. Connect the USB cable between the Action Replay and the PC
  4. Insert the Action Replay into the GBA slot of your NDS console (NOTE: also works if you have a spare GBA console laying around)
  5. Leave the switch in the DS position!
  6. Turn on the console and boot the GBA game.
  7. Hold down the Left shoulder buttons when it is starting. Do this when the "GameBoy" logo appears. It will force the Action Replay into USB mode.
  8. You should see the "GameBoy" logo appear first (if not try again or check cartridge connections)
  9. The Action Replay should in "USB SERVER MODE" (shown at the top of the screen)

  10. On your PC, run the "Nintendo DS - GBA Max Drive" program.
  11. For which device to manage, pick "Max Drive DS (DS)".
  12. Then pick "Backup my game cart save".
  13. Then select "My PC database" and press "Next =>".
  14. Enter the game name (suggestion "ac" or "acww")
  15. Enter the save name (example "test1")
  16. Press "Next =>".

  17. If you get the error "Game Cart not present, please insert before continuing" then the Action Replay does not understand the game in the DS slot. See the overview page.
  18. If you see the message "Warning! - Expected Flash version 1.02 but got 1.04 please make sure your console flash is upto date." - press OK and ignore it.
  19. If it starts the download process, it will take around 2 or 3 minutes to backup the game cartridge. If it successfully finished, skip to Step#5. You are lucky and don't need to do any modifications to the Action Replay unit.
  20. If you get other problems, it is typically the USB setup. Please read the manual, and try again.
NOTE: do not backup the the "Max Drive Vault" (the storage space on the Action Reply unit). Typically there won't be enough room since the AC:WW saves are big, even when compressed (30->100KB typically).

Uncompress a compressed game save to a DUC file (Export from AR)

  1. On your PC, run the "Nintendo DS - GBA Max Drive" program.
  2. For which device to manage, pick "Max Drive DS (DS)".
  3. Then pick "Manage and swap saves"
  4. Then select "My Max Drive vault" (you won't be touching the 'vault')
  5. If it says the "Console not connected" on the right-hand-side -- ignore it
  6. On the left-hand-side, select the game save (example "test1" under "acww")
  7. Press the Decompress button
  8. Select a folder to store the gamesave (Example "C:\", file will be saved there, eg: "C:\test1.duc")

If your software is missing the Decompress/Compress buttons, then update the AR software from the codejunkies website.

Compress a DUC file to a compressed game (Import to AR)

  1. The reverse of the above step.
  2. Keep the program running and in the "Manage and swap save" for "My Max Drive vault"
  3. TIP: on the left-hand-side, select and DELETE the old game save (example "test1" under "acww"). This will prevent the name from appearing in the list twice.
  4. Press the Compress button on the left-hand-side
  5. Select the file to compress
  6. Notice it appears in the left-hand-side list (this is the name you will use in the next step to restore)

Restore a backup from your PC to the AC:WW game cartridge

  1. Be sure the Action Replay is plugged in, turned on and the ACWW cartridge is in the "DS" slot of the unit.
  2. If not already running, run the "Nintendo DS - GBA Max Drive" program.
  3. If already running press the "<= Back" button until you get to the question "What would you like to do?"
  4. Pick "Restore my game cart save".
  5. Then select "My PC database". It may report the compression version.
  6. Select the gamesave to restore from the left-hand-list.
  7. Press "Next =>".
  8. Wait for the transfer (PC program should give you status)
  9. When transfer is complete, wait for the NDS to finish (it will report "Device Busy")
  10. When the "Device Busy" message is complete, you can turn off the console and move the ACWW cartridge from the Action Replay "DS" slot to the real cartridge slot on the top of the NDS (and play the game as usual)

  11. If you get the error "The console is not connected", then check the instructions. Power down everything (including the PC), reboot and try again.
