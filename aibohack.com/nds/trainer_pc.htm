Nintendogs Trainer for Windows (PC save editor)


USE AT YOUR OWN RISK.

Chiuahua on Steroids Also see the main page for more pictures of unnatural things you can do to your dogs There is a homebrew program that runs on the NDS.
This page describes the PC version (program runs on the PC)

Hardware Requirements

  • Any current version of Nintendogs (any of the three main versions, English or Japanese, or the Best Friends edition)
  • Any DS cart that can save and restore 256KB (2Mb) game saves to/from your PC, including:
    • Action Replay (open the .dss file)
    • SuperCard CF/SD (open the .sav file)
    • ETool
    • Compressed Game Save from my Homebrew Nintendogs Trainer
    • Perhaps other cards (if they save all 256KB)
  • A Windows PC to run the program.

PC Trainer Features

  • NOTE: the trainer doesn't have any built-in backup features. Since you already can save the Nintendogs game save, you have a backup.
  • NOTE: Save a copy of the original backup.

  • Easily add money or trainer points (become a stinking rich "Dog Master" in seconds)
  • Quicky switch the room style (for free, no waiting for the interior designer)
  • Reports number of items you've found, and how many you are missing
  • Easily add all the missing items, including
    • all the objects (toys, sports items, care items, music, etc). 100 total.
    • all the accessories. 60 total (including all the strange hats and collars)
    • all the room styles (9 of them) and clocks (8 of them). Room styles come pre-paid.
    • all the puppy breeds (20 regular ones)
  • NOTE: All objects/items available with any version of Nintendogs Cartridge. Some breeds don't work with the Japanese version.

  • Tweek the attributes of any of your dogs:
    • Breed - change the breed to any of the 20 available breeds (and two incomplete breeds - Papillon and Maltese - which look like beagles ;-)
    • Gender - quick sex-change
    • Color - change color (depends on breed)
    • Size - make dogs larger or smaller (or invisible or outrageously big)
    • NOTE: you can edit any of your dogs (active or in hotel, max of 8)

Pre-Requisite - learn how to backup/restore your game save

If you don't have the hardware to backup/restore game saves, or don't know how to use your hardware, please learn that first. Homebrew NDS hardware can be tricky (and expensive and primarily for "warez"). Every piece of hardware is different. After you figure it out, save the full backup of your Nintendogs game save somewhere permanent. The file should be 256KB uncompressed or slightly larger.

CONVERT NOTE: if using a homebrew setup and the Windows trainer gives you a load error, please use Shuny's converter at this external link. Converting it to an Action Replay file before editing it. Convert it back to the format used by your homebrew card after editing.

Installation/Usage Instructions

USE AT YOUR OWN RISK.
WARNING: backup your Nintendogs game save to your PC first.
  • Download the Nintendogs Trainer for Windows trainer_pc.zip. Run this program on your Windows PC (DogTrainW.exe)
    You don't need to install any custom software on your NDS.

  • Open the game save file (common extensions are ".sav", ".bin" or ".dss")
  • NOTE: if using the newer Action Replay, uncompress the gamesave first, then type in "*.DUC" and select the uncompressed .DUC gamesave.
  • If it fails to load see CONVERT NOTE above.
  • Change the settings as you see fit.
  • When done, press the "SAVE" button and it will update the game save file.
  • Install that updated game save in your NDS (steps depend on your hardware) and run the Nintendogs game.

General User Interface

The user interface is very simple.
  • In the upper left are owner attributes you can change (add money, trainer points, or switch the room style)
  • In the middle is the list of available dogs. Select one and you can view and edit the Dog Properties on the right hand side.
  • Note: if you change the breed/color, the trainer will report if the combination may cause problems.
  • At the bottom left is a summary of the items found. If you press the "Add Missing" it will add all known items.
  • Be sure to press "SAVE" to save the changes to the game save file (and copy it to your NDS before playing)
