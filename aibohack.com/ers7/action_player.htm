ActionPlayer (requires LAN)


If you have the AIBO LAN setup and an RCODE 7 personality on the memory stick, it can be remote controlled via ActionPlayer. This is similar to the ERS-2x0 AiboRemote, but without video and audio features. In addition to making AIBO walk, it can be used to play back any of the canned RCODE sticks.

ActionPlayer will to turn your AIBO into a remote control toy.

Quick Instructions

  • You need a PMS memory stick with initialized with an RCODE (ERS-7) personality. If you haven't done this before, I recommend starting with the Cylon 7 personality
  • The LAN settings must be configured for the PMS/RCODE stick. The easiest way to do this is to copy the WLANCONF.TXT file from your AiboMind stick (\OPEN-R\SYSTEM\CONF\WLANCONF.TXT) You must know the IP address of your AIBO.
  • Download ActionPlayer701.zip, unzip and run ActionPlayer.exe.

Usage

  • Enter the IP address of your AIBO and connect. If this fails, you probably have the WLANCONF.TXT configuration wrong.
  • When connected, the RCODE program will stop running.
  • The upper left will show PINK ball and BONE detection
  • The two distance sensors show the distance to obstacles in mm (upper number is sensor in the nose, lower number is chest 'cliff' sensor)
  • The Walk options will make AIBO walk.
  • The Look options will make AIBO move his head
  • The "Stand", "Sit", "Lie" buttons make AIBO move to those postures
  • Other postures
    • Pick other postures from the "Start Posture" dropdown list, and the bottom list will fill with possible actions.
    • Click on any of these actions, and AIBO will perform them. The action names are the same as you will find in YART7.
    • Most actions start and end with the same posture. Some change postures (at the top of the list with "POS CHANGE:"). If you perform these actions the list will change to that posture.
    • If that's not you want, or the list becomes empty, pick another "Start Posture" and try again.
