Lost-In-Space for ERS-7

"Lost In Space AIBO" is a TV themed personality for the ERS-7 created by Genie.

Run the personality on carpeting. The voice command "dance" puts Aibo into a sort of space walk to the Lost in Space Theme song, so carpeting is best. The personality has custom sounds and a few custom motions activated by various inputs...voice, sensor or other...


(artwork by Genie)

Supported AIBOs

  • - ERS-7 only.

Installation:

  • Dig up an RCODE/PMS memory stick (8MB or 16MB). It must be a PMS stick. It must be pink. It must not have an ERF- number on the sticker. You can't use the 32MB AiboMind pink stick.

  • Delete any files on the stick (do not format, use a Sony brand stick reader)
  • Download the RCODE7+YART memory stick image: RCODE7_YART703_E.zip (3.8MB)
  • Unzip the memory stick image on top of your RCODE/PMS stick.

  • Download this file lostinspace7.zip (1.4MB).
  • Unzip it to the root of the memory stick. It will ask you to replace some files. Answer "Yes".

  • Put the memory stick inside your AIBO and start it up.
  • Play with your AIBO on a carpet. Interact with your AIBO and make it "Dance"
