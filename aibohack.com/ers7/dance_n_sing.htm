Dance&Sing AIBO (for the ERS-7)

Dance&Sing AIBO is a simple game that makes your AIBO perform dance skits by itself. AIBO will play back music and dance.

Dance&Sing AIBO is installed using the AIBOCustomManager on the AIBOMind stick
AIBO Custom Manager compatible game pack (can easily be removed using the trashcan or replaced with another game). You can install it in any of the 3 game slots, but I recommend "Game Mode 2".

Supported AIBOs

  • - ERS-7 is supported with AIBOMind software [original or with SP1]

Required Hardware/Software:

  • An ERS-7 model AIBO
  • An AIBOMind stick (ERF-7AW01E) that came with the ERS-7

  • A Windows based PC
  • A memory stick reader (Sony brand stick reader strongly recommended)

  • You must have the free AIBO Custom Manager (ACM) installed on your PC.
  • You should be generally familiar with how the ACM works.
  • I strongly recommend you install the ACM program in the default folder on your C: drive:
    C:\Program Files\Sony\AIBO Custom Manager
    (it will make everything easier)

Installation

  • Download this self extracting ACD data file DanceNSing7_Collection1.exe (1.9MB)
  • Run it on your PC. If you have installed the ACM in the recommended folder on your C: drive, you can use the default unzip location.
  • After installation you don't need the DanceNSing7_Collection?.exe file/program anymore (it has installed the necessary ACM data onto your PC).

  • Run the AIBO Custom Manager (ACM) software. You should now see a new entry in the "Custom Data List" called "Dance&Sing (#1)"
  • Insert your AIBOMind memory stick into your PC memory stick reader.
  • Inside the ACM program:
    • Click the "Drive Path" to point to your memory stick drive.
    • You will see any existing customizations in the list on the left hand side.
    • Double click on the "Dance&Sing (#1)" custom data (in the right hand list)
    • It will ask you which number you wish to use (Game Mode 1, Game Mode 2 or Game Mode 3). You can pick any game slot, but I recommend Game Mode 2.
    • It should now appear in the left hand side under slot 2.

      NOTE: 'Dance&Sing (#1)' on the "2" slot is RED after it has been added but before it has been written to the memory stick

    • Press the "Write" button (bottom of the ACM) and click "OK" when asked
    • Wait while the memory stick is being updated
    • When complete, you normally don't need to eject the stick (on Windows XP)
    • Exit the ACM

  • Put the memory stick in your AIBO and start him up (see Usage instructions below).
  • NOTE: You may uninstall this dance pack at any time (use the trash can) or replace them with another dance pack. You may reinstall it later using the ACM.

Usage

The Dance&Sing is a standard game. To start the game, you can use the "Game Mode 2" card (Letter "M"), or use the voice guide mode (which is more reliable).

Starting Game Mode - voice guide mode

  • Press the front and rear back buttons at the same time (both white). Hold both for 3 seconds.
  • AIBO will lay down and say "Voice Guide Mode, top menu..."
  • Press the front back button (Blue) to go to the game picker.
  • Now press the middle back button (Orange) to pick Game #2.
  • This will enter the Dance & Sing game mode.

Using Dance&Sing Game Mode

  • When Dance&Sing starts, AIBO will say "Dance and Sing AIBO, press my head for instructions".
  • AIBO will situp, and the back buttons will light up (Blue, Orange, Red)
  • Now you can pick the dance you want by using the Blue, Orange and Red buttons. AIBO will speak the name of the song that can be performed. Extra long songs will mention how long the song takes to perform fully (you can abort it, see below)

  • Press 'Blue' (front back button) to go to the next category. There are three categories: Standing dances, Sitting dances, and Laying down dances. Each category has up to 10 dances. After you press Blue, it will say the current category, and then say the name of the first song ("Dances when standing up" is the category, "Joy to the World" is the first song)
  • Press 'Orange' if you don't want to see that dance, and select the next dance in that category.
  • Press 'Red' if you want AIBO to perform that dance.

  • When you press 'Red', AIBO will perform the dance. Don't touch any buttons until the dance is over and AIBO returns to the sitting position.
  • Also: To hear the short version of these instructions, press AIBO's head.

Exiting

  • When AIBO is dancing, do not press any back buttons.
  • If you press the 'Blue' (front back) button, AIBO will stop his current dance, eventually sit up and return to the song selection mode.
  • To exit the Game mode completely, press and hold AIBO's head for 3 seconds. AIBO will return to the normal AIBOMind autonomous mode.
  • If your AIBO jams or otherwise gets tilted (ear lights turn green or AIBO goes limp), press AIBO's head for 3 seconds.
  • NOTE: After a dance completes, AIBO will usually return to the main song picking mode, so you have to start from the beginning and pick next category and next song. If you stop the song early, it will usually return to the middle of the picking process (ie. just press 'Orange' to go to the next song).
