[AiboHack] AiboMind V3 Custom Audio

AiboMind V3 Custom Audio

Disclaimer: This is a test version.
Backup your AiboMind3 stick first! This will disable the existing male voice. To restore it you will have to recover from your backup. This version is not yet compatible with the future AiboCustomManager (V3).
Voices are synthesized (TTS). Not all TTS phrases make sense (eg: "HMMMM"). Sound effects in the talkative voice prompts are not included.
NOTE: Files are very large. Only download if you plan on using these files, and have an Internet connection reliable enough to download them.

Simplified Audio Customization Tool:
With the tool you can customize:
  • The main voice prompts including voice guide mode (examples: "Let's take a picture", "Voice guide mode, top menu")
  • The new talkative voice mode (examples: "I wanna play with the AIBONE", "I feel like I'm on vacation")
  • The 2 default games, demo/clinic mode and Simon (examples: "I will explain clinic mode", "I've seen bad, but you're worse")

The Custom Voices

  • AudioToolM3 tool - PC utility to merge custom audio [required]

  • Audrey: Synthetic Audrey voice (UK English) (preview first) -- after previewing download the big file Audrey M3 Voice (19.8 MB)

  • HeliumClaire: Synthetic Claire voice (US English) with effects (preview first) -- after previewing download the big file HeliumClaire M3 Voice (18.9 MB)

  • FranticLauren: Synthetic Lauren voice (US English) talking faster (preview first) -- after previewing download the big file FranticLauren M3 Voice (14.6 MB)

  • Charles: Synthetic Charles voice (UK English) (preview first) -- after previewing download the big file Charles M3 Voice (15.9 MB)

  • MiniAudrey: MiniAudrey voice - lower-quality version of Audrey and lower-quality SFX (fits on 32MB stick) (preview first) -- after previewing download the big file MiniAudrey M3 Voice (13.0 MB)

Instructions:

  • Backup your AiboMind3 memory stick !!! (use a Sony brand stick reader)

  • Download the AudioToolM3 program and any custom audio you want to use. Unzip them into the same folder (one .EXE, and multiple .ODA files). These files are big!
  • Place your AiboMind 3 memory stick into your (Sony brand) stick reader
  • Run AudioToolM3.exe
  • Select the drive of the memory stick (where there is a memstick.ind file)
  • NOTE: the first time AudioToolM3 is run, it will copy a 22.5MB file from your memory stick (may take a minute or two, depending on stick reader speed)
  • Pick one of the available voices for "Main". These are the main voice prompts. The default Male voice will be replaced.
  • Pick one of the available voices for "Talkative Voice". This is usually the same as the Main voice. These are the main voice used by AIBO when in Autonomous mode. The default Male voice will be replaced.
  • Pick on the the available voices for Game mode. This is usually the same as the Main voice. If you have installed custom games, leave the Game options alone. The custom game audio assumes the default game setup (Game slot #1=Feature Demo, Game slot #2=Repeat Me)

  • Press "Save" and it will replace the default audio with the voice mentioned.

Tech notes:
  • Audrey is the 16kHz Audrey voice from AT&T Natural Voices
  • Charles is the 16kHz Charles voice from AT&T Natural Voices
  • HeliumClaire is the 16kHz Claire voice from AT&T Natural Voices, with Helium and Underwater effects from CoolEdit 2K
  • TTS voices have ending silence trimmed, result normalized to 100% (using CoolEdit 2K) and compressed to 16kHz, 4bit-per-sample ADPCM (ERS-7 AIBO compatible)
  • MiniAudrey is the 16kHz voice downsampled to 8kHz (ADPCM). Canned sound effects are also reduced to make the whole thing fit in 32MB.

For synthetic Text-To-Speech (TTS) voices I recommend the AT&T Natural Voices package. It is significantly better than the free "Microsoft Mary" or "Microsoft Sam".
The better sounding voices are not free. See this website for a relatively cheap download (~$35 per voice with a one time cost of $35). NOTE: You do not need to buy it to use the canned voices above. However, if you use the TTS features of AiboEntertainmentPlayer often, you should consider a more natural voice (eg: synthesis feature, reading RSS feeds).

For WAV file editing, I still like the discontinued "CoolEdit 2K". It was a great free program (and an inexpensive $25 to remove some limitations). It was bought out by Adobe, repackaged and turned into $250 program that does essentially the same.
Demo versions of the tool can be found on the Internet. (if link fails, search for "ce2kmain.exe")
