[AiboHack] Aibo Mind Plus

Aibo Mind Plus

The "Life Plus" family supplements existing AiboWare to add interesting new features, most significantly LAN support. Through the Wireless LAN you can use AiboScope7 to observe what your AIBO is doing, in real time.

NOTE: unlike other AiboPet personalities, these are not RCode based, so they will not work on a RCODE/PMS stick. You must use an AiboMind memory stick (ERF-7AW01E)

Supported AIBOs

  • - ERS-7 is supported with AiboMind software [original or with SP1]

Required Hardware:

  • An ERS-7 model AIBO
  • An AiboMind stick (ERF-7AW01E) that came with the ERS-7
  • A memory stick reader to install the software (Sony brand recommended)
  • A Windows PC with a WiFi connection (for running AiboScope7)

Pre-Installation:

NOTE: AiboMindPlus is a small patch that is installed manually (not using the Aibo Custom Manager, yet). You must have the AiboMind software already installed and running on your stick.
  • Backup your AiboMind stick as a safety precaution.
  • Check the pink stick and make sure it says "ERF-7AW01E". AiboMind Plus will not work on any other stick.
  • The main feature of AiboMindPlus is WiFi LAN telemetry with AiboScope7. You should already have your AIBO configured for WiFi connectivity before you install AiboMindPlus patch. If you can't connect to AIBO's webpage to browse photos using WiFi, you should get that setup first. You don't need the AiboMail features working, since that is a more complicated setup.
NOTE: Installing AiboMindPlus will not alter the existing personality or other customized data on the stick.

Installation:

  • Put the stick into your computer's stick reader
  • Download AiboMindPlus701.zip (about 1/4 MB)
  • Unzip this file to the root of the memory stick. WinZip will ask you to replace existing files. Answer "Yes to all".
  • Put the memory stick inside your AIBO and start it up.
  • Play with your AIBO as you would regularly. Nothing has changed.
  • Install the AiboScope7 program.

Optional Installation Steps: PLUSCON

  • To tweek the features of your AIBO: run the \CUSTOM\PLUSCON.EXE program on your memory stick.
  • Only the camera settings can be changed, and this should be used in conjunction with CamCheck. However, there is not an ERS-7 version of CamCheck available (yet).
