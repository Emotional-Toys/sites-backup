[AiboHack] AiboMind V2 Custom Audio

AiboMind V2 Custom Audio

Simplified Audio Customization Tool:
NOTE: Files are very large. Only download if you plan on using these files, and have an Internet connection reliable enough to download them.

  • AudioToolM2 tool (V.03B, 64KB) - PC utility to merge custom audio [required, be sure you are using latest version]

  • Genie Full Voice (9.9 MB) Genie's voice. Thanks to Genie.
    (preview first)

  • Juliette Full Voice (8.2 MB) Synthetic Juliette voice (French language). Thanks to Dominique for the script.
    (preview first)

  • Audrey Full Voice (8.6 MB) Synthetic Audrey voice (UK English)
    (preview first)

  • BrianUK Partial Data (6.8MB) Partial data for 'the Englishman' (common Mind1 prompts) Thanks to Brian and X-Dog.
    (preview first)

  • Spanish Partial Data (6.4MB) Partial data for Spanish (common Mind1 prompts). Thanks to nihon.
    (preview first)

Instructions:

  • If you have installed custom games, restore the original games before installing custom audio. You can install the games again later (Game slot #1=Feature Demo, Game slot #2=Repeat Me)
  • Download the parts you want and unzip them into the same folder (one .EXE, and multiple .ODA files). These files are big!
  • Place your AiboMind 2 memory stick into your (Sony brand) stick reader
  • Run AudioToolM2.exe
  • Select the drive of the memory stick (where there is a memstick.ind file)
  • NOTE: the first time AudioToolM2 is run, it will copy a 10MB file from your memory stick (may take a minute or two, depending on stick reader speed)
  • Pick one of the available voices. Press "Save" and it will replace the default audio with the voice mentioned.
  • Optional: uncheck all the boxes, and press "Save" and it will restore the original voice.
  • Optional: check two boxes. For example if you pick one full voice (eg: Audrey) and one partial voice (eg: Genie), the common prompts will use the Genie voice, and the new Mind2 voice prompts will be Audrey.
