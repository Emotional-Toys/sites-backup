Cylon AIBO for ERS-7

"Cylon AIBO" is a Battlestar Galactica themed personality for the ERS-7 created by 'teamdax'. There are similar Cylon personalities for the ERS-2x0 and ERS-31x. See this page for the ERS-2x0/31x version.


(artwork by teamdax)

Supported AIBOs

  • - ERS-2x0/31x supported with older version.
  • - ERS-7 supported with this version.

Installation:

  • Dig up an RCODE/PMS memory stick (8MB or 16MB). It must be a PMS stick. It must be pink. It must not have an ERF- number on the sticker. You can't use the 32MB AiboMind pink stick.

  • Delete any files on the stick (do not format, use a Sony brand stick reader)
  • Download the RCODE7+YART memory stick image: RCODE7_YART703_E.zip (3.8MB)
  • Unzip the memory stick image on top of your RCODE/PMS stick.

  • Download this file cylon7.zip.
  • Unzip it to the root of the memory stick. It will ask you to replace some files. Answer "Yes".

  • Put the memory stick inside your AIBO and start it up.
  • Play with your AIBO. Follow the instructions (see below).

Usage instructions

The cylons are a race of robots- they destroyed their creators, and then tried to do the same to the human race. They almost succeeded, except for the Battlestar Galactica, and a few ship- who went off in search of the planet earth (for more information do a search for Battlestar Galactica

This ERS-7 version of Cylon is very similar to the 220 only version. It has the same boot up sequence, and performs an eye swipe when it scans. The responses to voice commands are also very similar.

Main differences

  • Cylon7 has some new sounds, and is certainly less impressed with the human race
  • Cylon7 can replay the theme song (by pressing BACK FRONT button)
  • Cylon7 can replay the introduction message (by pressing BACK MIDDLE button)
  • Cylon7 can not take photos, or register an owners name (these features are not supported in r-code 7)
  • Cylon 7 does not have the do nothing till head button pressed feature
  • Cylon is pretty active, He should wander around a lot, shooting everything, and threatening every human he finds. He will occasionally rest- this is the ideal time to give him verbal commands (all YART verbal commands have a specific response)
