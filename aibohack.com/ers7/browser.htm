ABrowser7

ABrowser7 for the ERS-7.
There are other browsers for earlier model AIBOs (ERS-110 & 111 and ERS-210, 220, 31x).

Installation

The AIBO Browser is installed on your PC. It can be used to browse an actual memory stick, or "memory stick images" which are stored on your PC.
To install:
  • Download the latest Browser: browser7.zip
  • Unzip it to a folder on your PC (say C:\AIBO\TOOLS). The program is called ABrowser7.exe

Startup

Before using the ABrowser7, I recommend you backup your memory stick (for a number of reasons).
When you run the ABrowser7.exe program, it will search for the first drive with a memory stick image on it. You can abort this process and manually pick the drive you want to examine. You must open a "MEMSTICK.IND" file at the root of the proper memory stick (or memory stick image). If you do not see a "MEMSTICK.IND" file, you may need to make hidden files visible in Windows (see "Tools", "Folder Options" and the "View" tab).

Main Interface



The main interface has a list on the left that you can select. This will pick on of the informational displays.
  • "Summary" gives a brief overview of the current status of your AIBO's personality
  • "Name/Owner" shows the recorded names in phonetic form. If you have not recorded your dog's name, it will report files missing. Record the name on AIBO using "Name Registration" and "Owner Registration" features
  • "LAN settings" shows the contents of the WLANCONF.TXT file
  • The "Advanced" features show even more lower level detail
  • The "Skill Levels" button (or menu) will let you change the skill level of your dog. The Maturity must be set to 100% before the other skill levels apply.
  • The "Volume" button will let you adjust the 4 preset volume levels
  • The "Human Faces" button (or menu) will show you the owner's face. If you see an entry starting with "face" then click on it. If you only see 8 "d0" .. "d7" faces, you need to perform the "Owner Registration" on AIBO and get it to remember your face.
  • The "View / "Color Preferences" menu will show the color preferences.

NOTE: the browser no longer shows photos. To see the photos taken by your dog, just open up the file "\OPEN-R\APP\PC\INDEX.HTM" on the memory stick.
