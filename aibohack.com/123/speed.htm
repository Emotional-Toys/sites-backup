[AiboHack] SpeedBoarder software

SpeedBoarder software

There are two versions of the SpeedBoarder Software If you purchased your SpeedBoard outside of Japan, it comes with the ERS-2x0 version of AiboWare. This works with either the ERS-210 or ERS-220 AIBOs. This version of the software runs on a pink PMS/RCODE stick (but is not written in RCODE). The proper stick, software and license to run the software comes with the SpeedBoarder.
There is an ERS-11x version of SpeedBoarder that is not available outside of Japan (at the current time). This works with the floppy eared ERS-110 or ERS-111 AIBOs. This version of the software runs on a pink or purple stick (8MB or 16MB). The program is written in a slightly improved version of RCode 11x.

If you have the Sound Controller (that comes with the ERS-110/111), you can use it to control either version of the SpeedBoarder software. For the ERS-11x, it is your only choice of control. For the ERS-2x0, the Sound Controller can be used in addition to the voice commands.

For general use instructions on the speedboard, voice commands (for the 2x0 version) and other warnings, please refer to the instructions that came with the SpeedBoard.

NOTE: the Japanese version of the software includes a "Power of Love" theme song. The English version does not.

Sound Controller

NOTE: The following sound controller commands will work with the 110/111 SpeedBoarder or 210/220 SpeedBoarder AiboWare. Not all commands have been tested on the 2x0 version.

I assume you are familiar with the basic operation of the Sound Controller.
Just like the normal ERS-11x AIBO, the SpeedBoarder will work on either channel "A" or "B". This lets you have two different AIBOs playing in the same area. The commands 80+Send (set to current channel), 81+Send (set to channel A), 83+Send (set to channel B), 84+Send (lower volume) and 86+Send (raise volume) work just like the normal 11x AiboWare.

Also adjust "VOL" on the remote so it is loud enough for AIBO to hear you. When AIBO is moving on the board or performing on the board, it will be difficult for him to hear the sound controller. Wait for him to stop moving before pressing the "SEND" button to send a command.

Game Mode

Normally you will use the sound controller in Game mode to move AIBO around. Press the "GAME" button on the controller when you first start. In this mode, you don't need to press the "SEND" button. The buttons are marked for what they do (not all work)
  • "2" move forward
  • "8" move backward
  • "1" move left wide turn
  • "3" move right wide turn
  • "4" move left sharp turn
  • "6" move right sharp turn

  • "*" (Lose) set to 'long' distance stride (speed up)
  • "#" (Win) set to 'short' distance stride (slow down)
  • "0" (Track) set to 'continuous' mode (carpet mode)

  • "7" (Pickup) Start recording
  • "9" (Release) Stop recording
  • "5" Playback recording

    Not Game Mode

    Press the "C" button to end the game mode. Now you can make AIBO perform specific skits:
    • 41 + SEND = AIBO will be in sound tracking mode (he will follow your voice). Repeat "41 + SEND" to exit this mode
    • 42 + SEND = Roulette
    • 43 + SEND = Change Voice (switch between English and Japanese)
