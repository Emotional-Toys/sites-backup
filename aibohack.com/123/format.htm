[AiboHack] Pink Memory Sticks for the ERS-7, ERS-2x0 and ERS-31X

Pink Memory Sticks for the ERS-7, ERS-2x0 and ERS-31X

Purple and Pink (or Blue and Red)

There are two colors of memory sticks used in AIBO, purple/blue and pink/red.

For the ERS-110/111 dogs, you will normally use blue (purple) memory sticks.
For the ERS-210 and later AIBOs, you will only use red (pink) memory sticks.

Note: there are also white memory sticks (used for "Magic Gate" audio). They are not recommended for use in AIBO.

ERS-7, ERS-210, 220, 311 and 312 Owners

For the ERS-210 and later AIBOs, you will only use red (pink) memory sticks. Pink sticks are only available from Sony (or other AIBO retailer).

These sticks are special "copy protected" sticks. This prevents you from copying the software from one stick to a different kind of stick.

You must exercise care when using these sticks, otherwise you will erase the copy protection, making the stick useless for your AIBO (it will turn it into the equivalent of a $25 purple memory stick).

Memory Stick Readers

The first warning is: ONLY USE SONY BRAND STICK READERS

Other brand stick readers may work, but some have been known to erase the copy protection, making the memory stick unusable. Stick with Sony brand memory stick readers to protect your ERS-210 memory stick investment.

DO NOT FORMAT

The second warning is: NEVER FORMAT A PINK MEMORY STICK

The recommended way to erase a pink memory stick:
  • Open the memory stick drive in Windows
  • Select any folders you see at the root (typically "OPEN-R" and sometimes "CUSTOM")
  • Delete the folders and their contents
  • Empty your recycling bin (if necessary)
This will remove the old files, perparing the stick for a new use.
There is usually a hidden file called MEMSTICK.IND at the root of the memory stick. You do not need to touch it (if you accidentally remove it, don't worry).

Quick Format only when needed

The warning about never formatting is only true about a full or unconditional format. If you corrupt a pink memory stick, you can "Quick" format it to restore the FAT file system. Usually you are lucky and the original corruption did not hurt the copy protection.

From Windows, this is a "Quick" option in the Formatting tool. From the command line, this is the "/Q" switch.
If you forget the "Quick" option, and perform an unconditional (full) format, you will probably erase the copy protection (bad).
Sometimes a "Quick" format may fail, and you will be asked whether to perform an unconditional (full) format. Do not do it (that would be bad).

ERS-110/111 Owners

ERS-110/111 owners have it easier. The purple sticks are not copy protected, and backup sticks can be purchased at almost any electronics or camera shop. Either 8MB or 16MB memory sticks will work.

Any brand stick reader will do but I still recommend the Sony brand readers, especially if you are thinking about buying an ERS-210 or later model AIBO in the future.

Newer ERS-110/111 AiboWare ("Party Mascot" and "Let's Play AIBO / CardDog") is released on pink colored sticks. These sticks are not copy protected, so you should think of them as blue sticks.
