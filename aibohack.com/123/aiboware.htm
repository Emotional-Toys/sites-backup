AiboWare Versions

This lists all the officially released AiboWare versions (most from Sony).
Prices are in US dollars. Japanese prices are typically lower. A $90 stick in the US costs 9000 Yen in Japan (close to $75 at current exchange rates).

See this page for a more technical analysis of the personalities and their complexity.

Recommendations and opinions on what to buy are my own. "220" section is UNDER CONSTRUCTION

ERS-110/111 AiboWare

ERS-11x AiboWare will work in the ERS-110 and ERS-111 dogs. It can be copied to any 8MB or 16MB memory stick. It is not copy protected. ERS-11x AiboWare is no longer being updated.

Core AiboWare (11x)

The core AiboWare grows up from a newborn to an adult. You did not have to purchase it separately since it that came with the robot hardware. This is like the "AiboLife" software used in later model robots (210 and later) which is sold separately from the hardware.

There are three released versions of the core AiboWare for the 110/111:
  • ERA-110 - original AiboWare for the ERS-110
  • ERA-111 "A" - AiboWare for the first batch ERS-111s
  • ERA-111 "B" - updated AiboWare for the second batch of ERS-111s

The recommended AiboWare is version ERS-111 "B". It has the most bug fixes and is slightly easier to train than the "A" version. You can download backup copies from AiboSite. Look under "Downloads" for "110 Virgin Memory Stick Images" (typically use the ERS-111B stick).

Optional AiboWare (11x)

There are three additional 11x AiboWare titles that are sold (or given away).
  • ERA-101 - "Hello AIBO" a demo personality (approx $90)
  • ERA-102M - "Party Mascot" ($90)
  • IP-0002J - Let's Play / Card Dog - a card playing personality that comes with a special set of cards (Japanese only, approx. $150)
Stick images of these software titles are not available on the Internet. The "Hello AIBO" stick is sold in Japan. It was given away as a promotional item in the US and UK for a limited period of time.

ERS-210 AiboWare

There are many versions of AiboWare for the ERS-210.
Each version of AiboWare has its own unique memory stick encryption key. You must purchase the proper red memory stick to use that version of AiboWare.
Most AiboWare comes in two languages (English and Japanese). AiboLife 1 and Hello AIBO (Type-A) are available in 4 languages (English, Japanese, French and German). Some AiboWare does not have voice commands, so language does not matter (eg: CardDog or Navigator).

Core AiboWare (2x0)

There are two main personality sticks that grow from a baby to an adult.
  • AiboLife 1 "W01" (ERF-210W01E/J/F/G) ERS-210 only, $90
  • AiboLife 2 "W01" (ERF-210W01E/J) ERS-210 and 220 only, $90
I recommend every AIBO owner have at least one AiboLife stick. ERS-220 owners should buy AiboLife 2. ERS-210 owners should buy AiboLife 1 and optionally AiboLife 2 as well (giving two different personalities).

Another core memory stick you should own is a $34 "Programmable Memory Stick", sometimes called a "PMS" or "RCODE" or "AMS" stick.

In comparison to the depth and complexity of the $90 "AiboLife" sticks, and the general usability of the $34 RCODE sticks, the other titles do not have the same depth or value. Therefore I consider them all "optional" depending on your interests, and your pocketbook. None of them are terrible, but when paying $90 to $150 for a stick, I have high expectations.

Optional AiboWare (210)

(NOTE: currently for the ERS-210. Will be updated with 220 compatibility information later)
  • Hello AIBO Type-A (ERF-210W02E/J/F/G) ERS-210, $80 Hello AIBO (Type-A) is recommended if you do not want to spend time raising your AIBO from a newborn. It acts like AiboLife but comes as a fixed personality adult ("Gentle Adult"). If you are comfortable performing "Brain Surgery", using the main AiboLife stick is recommended.
  • Party Mascot (ERF-210W03E/J) ERS-210, $80 Party Mascot has a few interesting games. It is more of a curiosity than a stick you will frequently use.

Optional LAN related AiboWare (210)

If you own the AIBO LAN setup, there are several additional AiboWare titles you should investigate. These products come with a pink memory stick containing the AiboWare and a CDROM to install the PC software. A PC running Windows is required.
  • AIBO Messenger (ERF-210W04E/J), $150 Email and web page reading AIBO. Requires PC and LAN.
  • AIBO Navigator (ERF-210W05E/J), $150 Remote control AIBO (requires PC and LAN). AIBO Navigator 2 to be released soon.

These are all good products, and focus on one aspect of the robot. The only negative is the price. $150 is a very expensive price for software, and the functionality isn't that new. For more information, check the main www.aibo.com site.

Japan-only Releases

  • Spring AIBO A special edition (demo) personality for the Spring 2001 color AIBOs (Japanese only)
  • Let's Play / Card Dog (IP-0001J, approx $150) A card playing personality that comes with a special set of color coded cards and a playing surface. Japanese only. ERS-11x version also available (see above).
  • Blue Adventure (NF-0001, approx $110) A read-along personality that comes with a book that AIBO "reads". Japanese only. An RCODE personality.
  • AIBO Planet 01 A talk-along personality given away to subscribers of the AIBOTOWN magazine. An extended version was on sale from AIBOTOWN (Japan). Japanese only. An RCODE personality.
For more information on Japanese release check out: HKora11's site (in Japanese of course).

ERS-311/312 AiboWare

The Latte and Macaron puppies have a limited number of AiboWare titles. Fortunately RCODE programs can run on the puppies to add to their capabilities.

Core AiboWare (31x)

  • AiboLife 310 (ERF-310W01E/J) ERS-31x only, $90
  • AiboPal 310 "E" (ERF-310W07E) ERS-31x only, $90
  • AiboPal 310 "J" (ERF-310W06J) ERS-31x only (Japan only)

Both sticks are recommended. Another core memory stick you should own is a $34 "Programmable Memory Stick", sometimes called a "PMS" or "RCODE" or "AMS" stick. This stick is the same as used in the 210 and 220 models.

The Japanese AiboPal ("W06J") has two distinct personalities, depending on whether you run it in Latte (white skin) or Macaron (black skin). The English AiboPal ("W07J") is color blind.
