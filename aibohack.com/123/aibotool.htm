AiboTool

AiboTool provides the most common Aibo examination and training features, in a small, easy to use program.
AiboTool works with all known AIBO models and all known memory sticks. The features available will vary depending on the memory stick type (ie. AiboLife has many customization features while RCODE has very few).

Platforms

AiboTool currently comes in two versions:
  • Windows version - runs under Windows 95, 98, Me, NT, 2K, XP.
  • CLIE version - runs under the Sony CLIE Palm PDA (color screen recommended). Now supports NX, NZ and UX models.

Installation

To Install Windows (PC) Version:
    Step 1) Be sure you can see hidden files in Explorer (check Folder Options)
    Step 2) Download the Windows version
    Step 3) Unzip it to your local hard drive
    Step 4) Run AiboTool.exe
    Step 5) Open "MEMSTICK.IND" file at root of memory stick

To Install CLIE (Palm) Version:

    Step 1) Download the CLIE version
    Step 2) Unzip it to the root of the AIBO memory stick
    Step 3) Place the memory stick in your CLIE - it will auto launch
    If the program does not auto-launch you can copy the program from the memory stick to your CLIE (or use MSAutorun for old CLIE models]

Sticks/Features Supported

Here are many features supported by AiboTool. Which features are available depend on the kind of AiboWare memory stick you have.
AiboTool will examine and report the name and version information for all currently known AiboWare sticks. For some sticks (like the "AiboLife" family), it will report detailed statistics.

AiboTool will let you change and customize your AIBO (through "Train"ing). Which features are customizable will depend on the AiboWare stick. For some sticks (life the "AiboLife" family), you can perform Brain Surgery or Speed up your AIBO's development. For other sticks, it may only let you change the color preferences.

The AiboTool features are the same for PC and CLIE versions with one exception. The "species" sound customization for ERS-11x AiboWare is not supported on the CLIE for size reasons. Display of photos on the CLIE platform requires a high resolution color display. The AIBO specific features are the same for the English and Japanese user interface.

    Feature Overview

    Examine

    The Examine shows a summary of the AIBO in the top window display. You can scroll this display to show more text.
    What is displayed depends on the type of AiboWare.

    Photographs (.IBO, .APH)

    Display the .IBO (AiboScope 11x) or .APH (AiboLife 1) photos found on the memory stick. On the CLIE version you can adjust the brightness and contrast of these photos.
    For newer software (eg: AiboLife 2, RCodePlus) photos are stored in standard JPEG format. Your computer should already have several ways of viewing those files. JPEG photo display is supported on newer model CLIEs.

    Language

    You can pick the English or Japanese User Interface. If your system does not have a DBCS font, you may see some strange characters.


    Train - Brain Surgery

    Brain Surgery changes the stage and personality variant of your AIBO.


    Train - Speed Up

    Speed Up changes the "Quality Time" spent at the current stage. Set this value to 100% to force the dog to evolve next time it boots. NOTE: only applies to sticks that evolve (like AiboLife).

    Train - Color Preference - ERS-11x

    Color Preference changes the way colors are interpreted by AIBO. You can rearrange the existing colors and how they are interpreted. You can not add your own colors.
    For the ERS-111, there are 4 colors that must be assigned:
    • BALL - usually PINK
    • Favorite (FAV) - usually BLUE
    • Hated (UNFAV) - usually ORANGE
    • 2nd BALL - usually YELLOW

    Train - Color Preference - ERS-210

    Color Preference changes the way colors are interpreted by AIBO. You can pick a default set of color preferences or customize them individually. You can not add your own colors.

    For the ERS-210, there are several default color configurations. If you customize the colors individually, there are 6 colors that must be assigned:
    • 1st Ball - usually RED
    • 2nd Ball - usually PINK
    • Flesh - usually FLESH
    • Favorite - usually BLUE
    • Hated - usually GREEN
    • 3rd Ball - usually PINK for Party Mascot

    You can use the ERS-210 browser (View / Color Preferences command) to see the color preferences of your AIBO.

    Train - Set Wake Time (ERS-2x0, 31x)

    For many of the general personality sticks (including all versions of AiboLife, HelloAibo and Explorer), you can set the wakeup time for your AIBO.
    A 24 hour clock is used. You can pick any time within 10 minute intervals. For example "20:10" (8:10 PM).

    NOTE: For AiboLife 1 owners who use the AIBO Fun Pack, you should use the Fun Pack feature instead.

    To use the WakeUp feature you must:
  • be sure your AIBO's internal clock is accurate
  • set the desired time using AiboTool
  • place the stick in your AIBO, and boot him up
  • When you want your AIBO to go to sleep,
    • For the ERS-210 and 220: pressing his chin and back for 3 seconds.
    • For the ERS-31x: roll his tail three times
  • AIBO will play a tune and go to sleep, but the green chest LED will slowly flash.
  • At the specified time, AIBO will wake up, and resume his normal personality

    If you press the pause button, AIBO will be turned off and will not wakeup by itself. If the chest LED turns off (and is not slowly flash) then your AIBO is turned off completely, and will not wake up at the given time.

    If you don't want your AIBO to wakeup at the given time, do not put him in sleep mode (just press his chest button as usual).

    Train - Change Gender (ERS-11x only)

    Change the way AIBO urinates.
    Choices include:
    • Female - squat
    • Male - raise leg

    Train - Change Species (ERS-11x only)

    Replace the dog-like "barking" sound with the sound of a different species.
    Choices include:
    • Dog
    • Cat
    • Pig
    • Lion
    • Human Baby
    • Pikachu
    • Sam Kinison
    NOTE: not supported on the CLIE version.
