Other Crap

All the good stuff is in the public releases (eg: YICT202 all-in-one release, and the ICSDK V2). There are a few other downloads that should only be used if you know what you are doing.

Documentation is minimal and support is minimal. For most things you should already be familiar with the public tools (YICT 202 and the ICSDK V2) before delving into this level.
  • walkup8a_orig.zip - cartridge personality that comes with the walk-up charger. Official SilverLit content, no hacks. No YICT support.
  • crominst11.zip - for installing CROM for a Super-ICybie if you have a purchased a Downloader.
  • ICAUD - command line utility for merging custom sounds into the generic personality. Follow the instructions closely.
  • yhsrc202.zip - C++ and assembler source to HIMAGE, YICT202 and ICAUD (not for the casual user).

    Due to lack of interest, this is essentially "abandon-ware" with no planned future improvements.
