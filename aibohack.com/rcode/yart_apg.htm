YART_APG - Audio Personality Generator

The "Audio Personality Generator" takes a number of .WAV files, and generates ACTIONs for them. These will appear in YART under "User Actions". The mouth movements (on the 210), the head movements and LED flashing are automatically generated for you.
Before using YART_APG you should be familar with simple WAV editing.

Usage

You should already have installed RCodePlus 2.51'A' on your memory stick as described here. Run the program "\CUSTOM\YART_APG" from the memory stick.

Check to make sure the drive letter for the memory stick is correct. You should change the "Audio Data Path" to point to a folder on your PC where you want to store your audio samples. All .WAV files in that folder must be the correct format.

WAV format

The WAV files you use must have simple names (start with a letter, and may only contain letters, numbers and "_").
This simple name will also be the Action name (ie. "FOO_BAR.WAV" becomes the RCODE statement "PLAY ACTION+ FOO_BAR")

WAVE files must be 8bit, 8kHz (8000Hz), mono.
I recommend CoolEdit 2000 as a good general purpose editor (free to try, $69 to buy).
