YART Advanced Topics

Advanced topics for your YART program.
YART is just a simple user interface for editing RCODE. Everything you do with YART can be done with a text editor by editing the file "\OPEN-R\APP\DATA\PC\AMS\R-CODE.R" on the memory stick.

Backup

Your YART program is stored in the "R-CODE.R" file in the "\OPEN-R\APP\PC\AMS" folder on the memory stick. You should copy this file to your PC to make a backup copy.

Share

After creating something cool with YART, you may want to share it with your friends. If the YART program uses no User Actions, you can email the "R-CODE.R" file to your friends. They need to install RCodePlus and YART as well, before they install your R-CODE.R file (in the "\OPEN-R\APP\PC\AMS" folder).

If you have created a more complicated personality, perhaps with custom audio, and you would like to publish an all-in-one installation (ie. easier for people to install), please send me an email.

Borrowing User Actions

To borrow user actions from other personalities, you need to install things in the right order. RCodePlus and the personality must be installed first, then the YART update. Follow the normal installation instructions for ObeyCat, DogsLife, or other personality.
Then install the YART update. This will replace the existing personality with a very simple one that you can customize. It will leave the User Actions intact for you to use (Pick the "User Actions" category).
