RCODE 1.2 (RcodePlus 2.50) "AiboSound" Tone Commands

"AiboSound"s - Background

In addition to the voice commands that AIBO can understand, there are several "aibosounds" that it can understand. These are short musical tunes.
For Aibo Life (1) and Hello AIBO, these tunes are used to detect ERS-11x AIBOs. For Party Mascot, these tunes are used for AIBO to AIBO communication. See the ERS-210 "tone command"s for more detail.

RCODE 1.2 AiboSounds

For RCODE 1.2, and the ERS-31x AiboPal software, these are used for both Aibo-to-Aibo communcation as well as for "Media Link" mode.
  • rcode01.mid - Start communicating with other AIBO
  • rcode02.mid - Enter Media link mode(1)
  • rcode03.mid - Enter Media link mode(2)
  • rcode04.mid - Enter Media link mode(3)
