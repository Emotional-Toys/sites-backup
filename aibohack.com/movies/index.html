AiboPet's Movie Site

PLEASE DOWNLOAD MOVIES ONLY ONCE

If you like a movie, please download it to your hard drive and run it from there.
If you download it, please use it. Do not download this entire site.

Here are selected movie clips of AIBO performances, using the Windows Media format (WMV).
If you have problems viewing Windows Media movies, please download the latest version of the free Windows Media Player.

First save the movie to your PC.
  • Right click on the video and "Save Target As..." (or "Save Link As" for Netscape users).
  • Save the .wmv file in a folder on your PC.
  • When complete open (ie. play) the .wmv file.
