EZ_DLC_Install V2

  • Tool for installing DLC items into a gamesave. Comes with official Nintendo DLC items, and many custom (overlapped) hacked DLC items. DISCLAIMER: THIS IS AN ADVANCED TOOL
    WARNING: This tool is for Wii owners who have WiiBrew installed (The Homebrew Channel), own a AC:CF disc, and have used "ACToolkit" before.

    Follow me on http://twitter.com/aibopet for update notifications when EZ_DLC_Install is updated with new DLC.

    Instructions

    Use "EZ_DLC_Install" to install the known 'DLC' downloadable content items. You run it like the other AC:CF gamesave tools like "ACToolkit", "NPC_Tool" and "PattView".
    • WARNING: Use at your own risk. ALWAYS keep an extra backup of the gamesave (RVFOREST.DAT)
    • Download the latest version here: ez_dlc_install_2.zip. Version WEEN 10 /10. I expect this to be the final version.
    • 'Extract' the gamesave from your Wii. (You should already know how to do this. The steps are the same as using the other tools like ACToolkit).
    • To be safe, make a backup of the RVFOREST.DAT file.
    • On your computer, run "EZ_DLC_Install" and open the RVFOREST.DAT
    • You can select the name of the items using your mouse and read about the DLC.


      (NOTE: screen shot image is out of date, new Nintendo items have been added)

    • To delete an installed DLC, simply uncheck the checkbox in front of it.
    • To add a DLC item, check it. You can only put one item in each 'slot'. For example if you want the "Guitar Axe1" you can't install any of the others in the "Chopping Weapons #1" slot.
    • BTW: Ignore the checkboxes in front of the categories.

    • Press the "Put in Pockets" button. This will replace the first items in your pockets/inventory with the new DLCs. This only works for Player#1.
    • Press "Save and Exit". It may warn you if it detects possible problems.
    • 'Install' that version back on your Wii
    • Run the game and play with the items in your pockets (you will have one of each)
