PMS sticks

A PMS is a Programmable Memory Stick, not just any pink memory stick. Also called a PMS/RCODE stick. You need a PMS stick to run RCODE, RCodePlus or OpenR SDK programs.

The old 8MB PMS stick ERA-MS008 stick is discontinued.
You can buy the 16MB PMS stick ERA-MS016 from Sony or RoboToys. For US owners it is available at sonystyle.com for $39.
  • ERA-MS016 PMS MemoryStick at sonystyle.com (US, $39).
    If you don't already have one, be sure to use a Sony brand memory stick reader/writer.
  • MSAC-USM1 Memory Stick Reader at sonystyle.com (US, $20).

    WARNING: ONLY USE SONY BRAND MEMORY STICK READERS!

    WARNING: NEVER FORMAT A PINK MEMORY STICK!

    WARNING: DO NOT USE THE 32MB AIBOMIND MEMORY STICK!

    WARNING: DO NOT USE THE 8MB AIBOLIFE MEMORY STICK!

    WARNING: FREE PROGRAMS NEED A 8 or 16MB PMS STICK (no ERF- number)!

    Ignore the "ERA-MS008" stamped on the back of the pink stick.
    Look at the front of the stick instead.
    If there is an "ERF-" model number printed on the sticker on the front, then it is NOT a PMS/RCODE stick. Purple/Blue sticks, white sticks or MS Pro sticks will not work either.

    PMS/RCODE sticks do not have an "ERF-" number, are always pink, and usually have no label or a home made label.
