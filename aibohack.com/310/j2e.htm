ERS-310 Japanese for English Speakers - OBSOLETE

NOTE: this page is obsolete.
You can use it if you like speaking to your AIBO in phonetic Japanese. Instead I recommend installing the "Engligh Vocab" patch and using the English voice commands.

This web page provides help for English speakers to control Japanese only AiboWare.
First find the proper guide for the stick you are using (use the ERF product number). Put the stick in your AIBO and boot him up.

To give a voice command to your Japanese AIBO:

  • Find the voice command you want to give in the last column ("english interpretation"). For example "shake paw" (number 3).
  • Do not say the phrase "shake paw" to your AIBO
  • Instead look at the third column, and pick one of the phonetic phrases, for example: 'a k u sh y u'.
  • Try pronouncing them based on how you think they will sound (for example: 'ahh koo shoe you').
  • If that doesn't work, try another pronunciation for the same voice command (semid), for example: 'o t e sh i t e' (oh tay she tay).

    TIP: Try to group them together in consonant + vowel pairs.
    TIP: Try pronouncing "/N/" like the letter 'n' or the letter 'm'.
    TIP: If you see strange symbols or numbers, you can ignore them.
    TIP: There is no letter L.

    Good luck.
