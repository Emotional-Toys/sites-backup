ERS-310 Japanese Language Voice Dictionary

This web page describes all the Japanese voice commands supported by AIBO in a detailed fashion.
Similar information is available for English.
NOTE: this page is not that useful. Japanese speakers already have a relatively complete vocabulary in the printed in struction manuals. For English speakers wanting to use some of the (relatively) newer 31x AiboWare, they should check out the Phonetic Japanese for imported memory sticks page.
