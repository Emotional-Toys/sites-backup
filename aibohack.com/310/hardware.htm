ERS-310 Hardware - CPU and memory

Check out hkora11's photos of the insides of the 31x.

CPU

The CPU is a Toshiba TMPR 4955A MIPS processor. It uses the MIPS IV instruction set, 32-bit data bus, and has a maximum speed of 200MHz. It runs at 192MHz.
NOTE: This is a different CPU than the one found in the ERS-2x0.

Memory

Dynamic memory is in two NEC D4518163 memory chips giving a total of 32MB of memory. This is SDRAM.
Read only memory is in single Toshiba TC58FVB321 flash memory chip giving a total of 4MB of flash ROM.

I/O Controller

There is a separate 8-bit micro-controller that performs I/O functions as well as controlling the main CPU (including powering it up and acting like a watch-dog). The chip is an NEC D780023. It is an 8-bit micro-controller with 24KB of ROM and 1KB of RAM, an analog-to-digital converter, a real-time-clock, a buzzer circuit (for the boot failure tune) and two serial ports (among other features).

Specs

Here are the main chips:
  • 200MHz MIPS CPU from Toshiba
  • 8-bit Micro-controller from NEC
  • 16MB SDRAM from from NEC
  • Toshiba 4M Flash ROM
Specs are easily found online. If you can't find a particular chip specific, please send me an email. There are additional custom Sony chips (see photos). They appear to be the same as the ERS-210 chips.
