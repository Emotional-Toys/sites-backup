Aph2Bmp - convert AIBO Photos to BMP format

Utility for owners of the ERS-210 and AiboLife (1).

Download Aph2Bmp

To install

  • Download and unzip the EXE somewhere on your computer.
  • Double click on any .APH file.
  • You will see the dialog "Open With..." dialog
  • Type in "Aibo Photo" or anything you like in the Description.
  • Then select "Other" and pick the "Aph2Bmp.exe" you downloaded and unziped.
  • Be sure the "Always use this program to open these files" is checked.

Now, whenever you double-click an APH file, it will create a similarly named .BMP file in that same directory.

After conversion, you can delete the .APH files if you like. What you do with the .BMPs is up to you.
