Flash Update for old ERS-210s

In order to use Open-R SDK programs on your ERS-210 AIBO, your AIBO has to have a newer version of OpenR (1.1.2 or 1.1.3). The original batch of ERS-210 AIBOs came with OpenR version 1.1.1 (old).

To update the flash memory in your ERS-210 you must use a program called "FlashUpdater for ERS-210". You can download this tool from the OpenR SDK website.

If you have one of the following AIBOs, there is no need to update the flash memory:
  • ERS-210 that has been updated via the "220 transform kit(ERS-220E1)"
  • ERS-220
  • ERS-210A
  • ERS-220A
  • Some but not all ERS-210 sold after Jan 2002. If in doubt, there is no compelling need to upgrade until you hit a snag with one of the newer programs.

Updating the Flash ROM changes the ROM inside of AIBO, so be sure to read and follow the instructions carefully. In general it is a positive change (fewer bugs, more programs work, fewer crashes and people anecdotally report that voice recognition gets better).

WARNING:
Compatibility with very old AiboPet-ware (pre-'Legit-i-Mutt') will be lost.
Compatibility with old RoboCup tools (pre-OpenR SDK) will be lost.

To perform the Update:

  • Register and login at the OpenR SDK site. If you have not previously registered, you must do so first. You will get a login name and a password and immediate access to the site.
  • Once you have logged in, select "Download", then "FlashUpdater for ERS-210(4MB)".
  • Pick either the US or non-US EULA, read it if you like, and then press the second "AGREE" button: "upgrade-OPEN_R-1.1.3-r2.exe (self extraction file for Windows) download" -- easier to use IMHO
  • Download and run the installer ("upgrade-OPEN_R-1.1.3-r2.exe")
  • When it asks for a folder, specify a folder on your PC (eg: "C:\AIBO"). It will create a sub-folder called "upgrade" (eg: "C:\AIBO\upgrade").
  • In that folder are two files ("README_E.TXT" and "README_J.TXT") and one folder ("OPEN-R"). Read the instruction in README_E.TXT

    Here is my version of the instructions
  • Find a currently unused pink PMS/RCODE stick (ie. it will not have an "ERF-" part number on it). Either 8MB or 16MB size will work.
  • Select any folders you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. Do not format the stick.
  • Drag the "OPEN-R" folder from the "C:\AIBO\upgrade" folder on your PC to the memory stick. When complete there will be an "OPEN-R" folder on the memory stick and nothing else (except perhaps an empty MEMSTICK.IND file).
  • Be sure your ERS-210 AIBO has a full battery.
  • Put the stick in your AIBO, boot him up
  • After about 10 seconds, he will start beeping while the upgrade is in progress. Wait for the beeping to stop (about 1 minute).
  • Do not remove the stick or the battery during the process!
  • If you have difficulties, refer to the troubleshooter topics in README_E.TXT
  • After the upgrade/update is complete, you should delete the files on the stick, and reuse it for other PMS/RCODE personalities. You will not need to upgrade that AIBO again.
