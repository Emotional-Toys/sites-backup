Scrap Yard Aibo - by teamdax

Scrap Yard Aibo is a demo personality for the 210, it will not run on the 220 or 31x.


(photos and usage instructions also by teamdax)

Supported AIBOs

  • - Only the ERS-210 is supported.

Installation

Required Hardware:

  • An ERS-210 model AIBO
  • A Pink Programmable Memory Stick (AKA "PMS" or RCode stick) - ERA-MS008 or ERA-MS016, $34->$39
  • A memory stick reader to install the software (Use only Sony brand stick readers)

Installation:

  • Put an available PMS stick into your computer's stick reader
  • Select any folders you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. Do not format the stick.
  • Download the proper RCodePlus runtime for the model of your AIBO.
    • For the ERS-210: RCode Plus 2.52 'A' (210)
    • (Save this .ZIP file on your computer. You may want to use it again later.)
  • Unzip this RCodePlus file to the root of your PMS memory stick.
  • Download the ScrapYard Personality: scrapyard210.zip (1.1MB)
  • Unzip that file on top of your RCODE/PMS stick (replacing any files it asks about).

  • Put the memory stick inside your AIBO and start it up.
  • Play with your AIBO. Follow the instructions (see below).

Usage

  • Simulated DHS: ScrapYard Aibo will exhibit simulated DHS at random intervals. The sound is pretty realistic (it was recorded from a real DHS head).
  • Taking a photo: if you want to take a photo just press (and release) the head button. ScrapYard Aibo will eventually sit down, and take a photo
  • Pause: if you want to pause scrape yard just press his back button. He will eventually sit and wait. When you want to un-pause him, press his head button.
  • Colour of tail: the tail will be orange. If you take a photo, it will turn (and stay) blue. If you want to turn it orange again, press the chin sensor.
  • TIP: voice command for simulated DHS: register the owner's name as "DHS" (by saying "owner registration" first). Afterward, every time you say "DHS" ScrapYard Aibo will exhibit simulated DHS.
    NOTE: getting scrape yard to respond to voice commands can be difficult.
  • Try all possible voice commands for amusing actions.
    Be sure to speak when Aibo is not moving.

WARNING: Scrap Yard is a pretty hacked personality, it can hang occasionally. If ScrapYard Aibo stops, try the following:
  • Try a verbal command (such as sit)
  • Try pressing the head button
  • Try tipping him.
    If none of the above work turn him off, and back on. This problem seems worse on a super core

Customization

You can customize the personality using YART. To run YART on the Stitch personality, just open the "\CUSTOM" folder on the memory stick and run the YART.EXE program in there.

Please refer to the general YART tutorial if you have never used YART before.
