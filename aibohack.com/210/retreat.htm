Re-Treat tool for ERS-220 owners

Do you have an ERS-220 but want to try an ERS-210 RCode based personality ?
Are you familiar with running DOS command line tools (ie. .BAT files with arguments) ?
If so, you should download the "RETREAT" tool.

NOTE: personalities from this site (AiboHack.com) are already converted to work with all model AIBOs that make sense (typically one install for the 210, 220 and 31x). Do not use RETREAT for AiboHack personalities.
You need to use RETREAT for personalities found elsewhere that have restrictive copyrights or licensing terms. This includes the "Treat" personalities from the US Aibo site (hence the name of the tool "RETREAT"). You must download the two separate pieces and run a simple batch file yourself. The resulting ERS-220 personality is for your own personal use only and can not be distributed (see the LICENSE.TXT file that comes with most personalities)

Required Hardware:

  • An ERS-220 model AIBO
  • A Pink Programmable Memory Stick (AKA "PMS" or RCode stick) - ERA-MS008 or ERA-MS016, $34->$39
  • A memory stick reader to install the software (Use only Sony brand stick readers)
  • A DOS/Windows PC computer attached to the Internet for downloading

NOTE: you should be familiar with installing personalities to the PMS/RCODE memory stick and be familiar with running DOS command line tools.

Installation

One time download/install

  • Download the ReTreat tools retreat.zip
  • Unzip them to a folder on your computer ( "C:\AIBO\RETREAT" is recommended )
  • Download the ERS-220 version of RcodePlus (or you can use standard RCODE 1.2). RCodePlus 2.52 (220) - for the ERS-220
  • Save the ZIP file to your PC ( "C:\AIBO" is recommended ). Do not unzip it yet.

Then for each RCode based personality you want to try:

  • Find the ERS-210 RCode personality you want to try, and download it. Examples that have been tested include: AiboTreat#2 (AiboBoo)
    AiboTreat#3 (Thanksgiving) - download the 210 version
    Other general ERS-210 RCode based programs should work
  • Unzip the ERS-210 program to your PC (not the memory stick). Pick a folder name you can remember ( "C:\AIBO\TREATx" where x = 2 or 3 is recommended ).
  • Put an available PMS stick into your computer's stick reader
  • Select any folders you see at the root of the stick (usually OPEN-R and CUSTOM) and delete them. Do not format the stick.
  • Unzip the RCodePlus runtime ( C:\AIBO\RCodePlus251A(220).zip ) to the top level (root) of the memory stick
  • Now the tricky part. Start a command shell and type: 
            C:
        cd \AIBO\RETREAT
        RETREAT12 F: C:\AIBO\TREATx
    Replace "F:" with the drive letter of your memory stick. Replace "C:\AIBO\TREATx" with the path where you installed the ERS-210 personality/treat. Adjust other path names if you did not use the recommended directory names.

  • If successful, the memory stick will now contain an ERS-220 compatible version of the personality/treat.
  • Put the memory stick inside your AIBO and start it up.
  • Play with your AIBO. Follow the instructions included with the personality/treat.

    DISCLAIMER: if you find the personality is lame - don't blame me.

    NOTE: there is also a "RETREAT13" batch file that converts ERS-210 to ERS-31x personalities (needed for Aibo Boo only)
