AiboRemote (requires LAN)


If you have the AIBO LAN setup, RCodePlus based personalities (including ObeyCat and DogsLife) can be remote controlled via AiboRemote.

AiboRemote will to turn your AIBO into a remote control toy.
  • Download AiboRemote252.zip, unzip and run AiboRemote.exe (requires a LAN and an AIBO running RCode Plus)

Problems?

AFAIK: all the nagging LAN crashing problems have been fixed.
If your Aibo shuts down or you are having audio capture problems, be sure you are using the latest release of RCodePlus (versions prior to 2.52 'A' had intermittent LAN problems).
