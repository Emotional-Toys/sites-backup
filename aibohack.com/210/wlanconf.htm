WLANCONF.TXT

If you wish to use Aibo Scope LIVE!, or AiboRemote, you will need a LAN setup. You should copy the WLANCONF.TXT file from a stick you have already configured to the AiboLifePlus stick. This file is found in the "\OPEN-R\SYSTEM\CONF\WLAN\HTML" folder.
