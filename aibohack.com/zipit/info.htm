Zipit general device info

Hardware

BIG CHIPS:
  • Cirrus EP7312-CR-90 (ARM based CPU running up to 90MHz)
  • Agere WL600114LY (WiFi controller and shielded radio)
  • Hynix HY5V26D (16MB SDRAM, 16 bits wide)
  • MX 29LV160ATxBC-70 (2MB Flash ROM)
  • LPC915 (8 bit uP)
  • WM8751 (stereo DAC)
Other interesting hardware:
  • WiFi (802.11b) capable with adequate built-in antenna
  • 320x240 grey-scale LCD (16 levels of grey)
  • Thumb-style keyboard, with a few special keys
  • Stereo headphones connector, with extended pins (perhaps for remote control) - hooked to stereo DAC

Software (aka Firmware)

The 2MB of Flash ROM contains the software/firmware for running the device. It contains a version of ARM Linux as well as a single program that provides all the main features (User Interface, WiFi connection, IM, future audio streaming etc). Source patches for the Linux kernel and some device drivers is available on the official website.

Firmware is updated automatically. The Zipit program checks for newer versions of the official firmware from a public website. This feature can be disabled or reused to upload your own tested firmware.

The software reflashing option supported on this website lets you use that automatic update mechanism to install a more general version of Linux, including login and telnetd services.
